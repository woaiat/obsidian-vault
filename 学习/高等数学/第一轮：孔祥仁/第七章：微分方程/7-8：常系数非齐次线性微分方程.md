## 一、 形式
$$y'' + py' + qy = f(x) \neq 0$$
**通解结构：** 非齐通 + 非齐特 ($y = Y + y^*$)

---

## 二、 第一类：$f(x) = e^{\lambda x} \cdot P_m(x)$
其中 $P_m(x)$ 是关于 $x$ 的 $m$ 次多项式。

**推导过程：**
设特解为 $y^* = e^{\lambda x} R(x)$，则：
$y^{*'} = e^{\lambda x} [\lambda R(x) + R'(x)]$
$y^{*''} = e^{\lambda x} [\lambda^2 R(x) + 2\lambda R'(x) + R''(x)]$

代入左端（LHS）：
$e^{\lambda x} [R''(x) + (2\lambda + p)R'(x) + (\lambda^2 + p\lambda + q)R(x)] = e^{\lambda x} P_m(x)$

**(1) 若 $\lambda$ 不是 $r^2 + pr + q = 0$ 的根：**
即 $\lambda^2 + p\lambda + q \neq 0$。不难设 $R(x) = R_m(x) = b_0 x^m + \dots + b_{m-1}x + b_m$。
代入上述方程，由于系数对应，可求解得 $R_m(x)$。
此时特解：$y^* = R_m(x) e^{\lambda x}$。

**(2) 若 $\lambda$ 是 $r^2 + pr + q = 0$ 的单根：**
即 $\lambda^2 + p\lambda + q = 0$，且 $2\lambda + p \neq 0$。则 $R'(x)$ 必为 $m$ 次多项式。
设 $R(x) = x R_m(x)$，代入上述推导同理可解得 $R_m(x)$。
此时特解：$y^* = x R_m(x) e^{\lambda x}$。

**(3) 若 $\lambda$ 是 $r^2 + pr + q = 0$ 的重根：**
即 $\lambda^2 + p\lambda + q = 0$，且 $2\lambda + p = 0$。设 $R(x) = x^2 R_m(x)$，同理得：
此时特解：$y^* = x^2 R_m(x) e^{\lambda x}$。

**综上，特解形式为：$y^* = x^k R_m(x) e^{\lambda x}$ ($k$ 为重根数)。**

---

### 例1. 求 $y'' - 2y' - 3y = 3x + 1$ 的通解。
**解：**
1. 齐次通解：特征方程 $r^2 - 2r - 3 = 0 \Rightarrow (r-3)(r+1) = 0 \Rightarrow r_1=3, r_2=-1$。
   故 $Y = C_1 e^{3x} + C_2 e^{-x}$。
2. 求特解 $y^*$：$\lambda = 0, P_1(x) = 3x + 1 \Rightarrow \lambda \neq r_{1,2} \Rightarrow k = 0$。
   设 $y^* = e^{\lambda x}(ax + b) \cdot x^0 = ax + b$。
   $y' = a, \quad y'' = 0$。
   代入方程：$0 - 2a - 3(ax + b) = -3ax - (2a + 3b) = 3x + 1$。
   对照系数：$\begin{cases} -3a = 3 \\ -(2a + 3b) = 1 \end{cases} \Rightarrow \begin{cases} a = -1 \\ b = \frac{1}{3} \end{cases}$。
3. 得 $y^* = -x + \frac{1}{3}$。
**通解：** $y = C_1 e^{3x} + C_2 e^{-x} - x + \frac{1}{3}$。

---

### 例2. 求通解：$y'' - 5y' + 6y = xe^{2x}$
**解：**
1. 齐次通解：$r^2 - 5r + 6 = 0 \Rightarrow (r-2)(r-3) = 0 \Rightarrow r_1=2, r_2=3$。
   故 $Y = C_1 e^{2x} + C_2 e^{3x}$。
2. 求特解 $y^*$：$\lambda = 2, P_1(x) = x \Rightarrow \lambda$ 是单根 $\Rightarrow k = 1$。
   设 $y^* = e^{2x}(ax + b)x = e^{2x}(ax^2 + bx)$。
   $y^{*'} = e^{2x} [2ax^2 + (2a+2b)x + b]$
   $y^{*''} = e^{2x} [4ax^2 + (8a+4b)x + 4b + 2a]$
   代入 LHS：$e^{2x} [-2ax + 2a - b] = e^{2x} \cdot x$。
   对照系数：$\begin{cases} -2a = 1 \\ 2a - b = 0 \end{cases} \Rightarrow a = -\frac{1}{2}, b = -1$。
   故 $y^* = e^{2x}(-\frac{1}{2}x^2 - x)$。
**通解：** $y = (-\frac{1}{2}x^2 - x + C_1)e^{2x} + C_2 e^{3x}$。

---

### 补充例1. 求通解：$y'' - 2y' + y = (3x+2)e^x$
**解：**
1. 齐次：$(r-1)^2 = 0 \Rightarrow r_1=r_2=1$。故 $Y = (C_1 + C_2x)e^x$。
2. 特解：$\lambda = 1, P_1(x) = 3x+2 \Rightarrow \lambda$ 是重根 $\Rightarrow k = 2$。
   设 $y^* = e^x(ax+b)x^2 = e^x(ax^3 + bx^2)$。
   $y^{*'} = e^x [ax^3 + (3a+b)x^2 + 2bx]$
   $y^{*''} = e^x [ax^3 + (6a+b)x^2 + (6a+4b)x + 2b]$
   代入 LHS：$e^x [6ax + 2b] = (3x+2)e^x$。
   对照系数：$6a = 3 \Rightarrow a = \frac{1}{2}, 2b = 2 \Rightarrow b = 1$。
   故 $y^* = e^x(\frac{1}{2}x^3 + x^2)$。
**通解：** $y = (\frac{1}{2}x^3 + x^2 + C_2x + C_1)e^x$。

## 三、 第二类：$f(x) = e^{\lambda x} [P_l(x)\cos\omega x + Q_n(x)\sin\omega x]$
($P_l(x)$ 和 $Q_n(x)$ 不同时为 0)

**推导过程（利用欧拉公式）：**
$\cos\omega x = \frac{1}{2}(e^{i\omega x} + e^{-i\omega x}), \quad \sin\omega x = \frac{1}{2i}(e^{i\omega x} - e^{-i\omega x})$。
代入 $f(x)$：
$f(x) = e^{\lambda x} [P_l(x) \frac{e^{i\omega x} + e^{-i\omega x}}{2} + Q_n(x) \frac{e^{i\omega x} - e^{-i\omega x}}{2i}]$
$= (\frac{1}{2}P_l(x) + \frac{1}{2i}Q_n(x)) e^{(\lambda + i\omega)x} + (\frac{1}{2}P_l(x) - \frac{1}{2i}Q_n(x)) e^{(\lambda - i\omega)x}$

设 $P(x) = \frac{1}{2}P_l(x) - \frac{1}{2i}Q_n(x)$，则 $\overline{P}(x) = \frac{1}{2}P_l(x) + \frac{1}{2i}Q_n(x)$。$m = \max\{l, n\}$。
1. $y^* = x^k R_m(x) e^{(\lambda + i\omega)x}$ 为 $y'' + py' + qy = P(x)e^{(\lambda+i\omega)x}$ 特解。
2. $\overline{y}^* = x^k \overline{R}_m(x) e^{(\lambda - i\omega)x}$ 为 $y'' + py' + qy = \overline{P}(x)e^{(\lambda-i\omega)x}$ 特解。

**原方程特解为：**
$y^* = x^k e^{\lambda x} [R_m(x) e^{i\omega x} + \overline{R}_m(x) e^{-i\omega x}]$
$= x^k e^{\lambda x} [R_m(x)(\cos\omega x + i\sin\omega x) + \overline{R}_m(x)(\cos\omega x - i\sin\omega x)]$
$= x^k e^{\lambda x} [R_m^{(1)}(x)\cos\omega x + R_m^{(2)}(x)\sin\omega x]$，($m = \max\{l, n\}$)。
（$k$ 为 $\lambda \pm \omega i$ 是否为特征根，故 $k \leq 1$）

---

### 例3. 求 $y'' + y = x\cos 2x$ 的通解。
**解：**
1. 齐次：$r^2 + 1 = 0 \Rightarrow r = \pm i$。故 $Y = C_1\cos x + C_2\sin x$。
2. 特解：RHS $= e^0 [P_1(x)\cos 2x + Q(x)\sin 2x]$。$P_1(x)=x, Q_0(x)=0, k=0$。
   设 $y^* = (ax+b)\cos 2x + (cx+d)\sin 2x$。
   $y' = (2cx + 2d + a)\cos 2x + (-2ax - 2b + c)\sin 2x$
   $y'' = (-4ax - 4b + 4c)\cos 2x + (-4cx - 4d - 4a)\sin 2x$
   代入：$(-3ax - 3b + 4c)\cos 2x + (-3cx - 3d - 4a)\sin 2x = x\cos 2x$。
   $\begin{cases} -3a = 1 \\ -3b + 4c = 0 \\ -3c = 0 \\ -3d - 4a = 0 \end{cases} \Rightarrow a = -\frac{1}{3}, b = 0, c = 0, d = \frac{4}{9}$。
**通解：** $y = C_1\cos x + C_2\sin x - \frac{1}{3}x\cos 2x + \frac{4}{9}\sin 2x$。

---

### 补充例1. 求通解：$y'' + 2y' + 2y = e^{-x}\cos x$
**解：**
1. $(r+1)^2 = -1 \Rightarrow r = -1 \pm i$。故 $Y = (C_1\cos x + C_2\sin x)e^{-x}$。
2. 特解：$\lambda = -1, \omega = 1, \lambda \pm \omega i = -1 \pm i$ 是特征根 $\Rightarrow k = 1$。
   设 $y^* = e^{-x}(a\cos x + b\sin x)x$。
   $y' = e^{-x} [((b-a)x + a)\cos x + ((-b-a)x + b)\sin x]$
   $y'' = e^{-x} [((a-b)x - a + (a+b)x - b)\sin x + ((a-b)x - a - (a+b)x + b)\cos x]$
   $\Rightarrow y'' = e^{-x} [(-2bx + 2a - 2b)\cos x + (2ax - 2b - 2a)\sin x]$
   代入 LHS：$e^{-x} [2b\cos x - 2a\sin x] = e^{-x}\cos x$。
   $b = \frac{1}{2}, a = 0 \Rightarrow y^* = e^{-x} \cdot \frac{1}{2}x\sin x$。
**通解：** $y = (C_1\cos x + C_2\sin x)e^{-x} + \frac{1}{2}x\sin x e^{-x}$。

---

### 补充例2. 求通解：$y'' + 2y' + 2y = 2e^{-x} \cos^2 \frac{x}{2}$
**解：**
RHS $= e^{-x}(1 + \cos x) = e^{-x} + e^{-x}\cos x$。
故分别求 LHS $= 0 通解，再求 LHS $= e^{-x}\cos x$ 和LHS $= e^{-x}$ 特解（叠加原理）。
1. 齐次通解已求。
2. LHS $= e^{-x}\cos x$ 的特解由“补充例1”知：$y_2^* = \frac{1}{2}x\sin x e^{-x}$。
3. 求 LHS $= e^{-x}$：$\lambda = -1, k=0$。
   设 $y_1^* = e^{-x} \cdot a$。
   $y' = e^{-x}(-a), \quad y'' = e^{-x}(a)$。
   代入：$e^{-x} \cdot a = e^{-x} \Rightarrow a=1, y_1^* = e^{-x}$。
**综上所述：** $y = (C_1\cos x + C_2\sin x + \frac{1}{2}x\sin x + 1)e^{-x}$。

---

## 总结
1. $f(x) = e^{\lambda x} \cdot P_m(x)$，设 $y^* = x^k Q_m(x) e^{\lambda x}$。
2. $f(x) = e^{\lambda x} [P_l(x)\cos\omega x + Q_n(x)\sin\omega x]$。
   设 $y^* = x^k e^{\lambda x} [R_m^{(1)}\cos\omega x + R_m^{(2)}\sin\omega x]$。
   ($k$ 为 $\lambda + \omega i$ 重根数，$m = \max\{l, n\}$)。
