# 笔记：7-2 可分离变量微分方程

![封面图占位符：丛林中探出头的小熊、企鹅和狐狸，文字为 SPRING](image_placeholder_page1.png)

## 一、形式
$$g(y)dy = f(x)dx$$

## 二、解法
对等式两端同时积分：
$$\int g(y)dy = \int f(x)dx \Rightarrow G(y) = F(x) + C$$

---

### 例 1：求通解 $\frac{dy}{dx} = 2xy$

1. **当 $y = 0$ 时**：
   $\frac{dy}{dx} = 0 = 2x \cdot y \Rightarrow 0 = 0$。
   因此，$y = 0$ 是方程的解。 $\checkmark$

2. **当 $y \neq 0$ 时**：
   分离变量：$\frac{1}{y}dy = 2xdx$
   两边积分：$\int \frac{1}{y}dy = \int 2xdx \Rightarrow \ln|y| = x^2 + C_1$
   解出 $y$：$|y| = e^{C_1} e^{x^2} \Rightarrow y = C e^{x^2} \quad (C \neq 0)$

**综合以上：** 通解为 $y = C e^{x^2}$（$C$ 为任意常数）。

### 补充例题

#### 补例 1：求通解 $y' = \frac{y}{x}(1-x)$
分离变量：$\frac{1}{y}dy = (\frac{1}{x} - 1)dx$
积分：$\ln y = \ln x - x + C_1$
结果：$y = C x e^{-x}$

#### 补例 2：$xy' + y = 0$，满足 $y(1) = 1$ 的特解
分离变量：$-\frac{1}{y}dy = \frac{1}{x}dx \Rightarrow -\ln y = \ln x + \ln C$
代入初值：将 $y=1, x=1$ 代入，得 $0 = 0 + \ln C \Rightarrow C = 1$。
代回原式：$-\ln y = \ln x \Rightarrow \ln \frac{1}{y} = \ln x$
特解：$y = \frac{1}{x}$

#### 补例 3：求通解 $(e^{x+y} - e^x)dx + (e^{x+y} + e^y)dy = 0$
提取公因式：$e^x(e^y - 1)dx + e^y(e^x + 1)dy = 0$
分离变量：$\frac{e^y}{1-e^y}dy = \frac{e^x}{1+e^x}dx$
积分：$-\int \frac{d(1-e^y)}{1-e^y} = \int \frac{d(1+e^x)}{1+e^x}$
$\Rightarrow -\ln(1-e^y) = \ln(1+e^x) + \ln C_1$
结果：$(1-e^y)(1+e^x) = C$

---

#### 补例 4：通过积分方程求函数 $f(x) = \int_0^{2x} f(\frac{t}{2})dt - \ln 2$
对 $x$ 求导：
$f'(x) = f(\frac{2x}{2}) \cdot 2 \Rightarrow f'(x) = 2f(x)$
转化为微分方程：$\frac{1}{y}dy = 2dx$
积分：$\ln y = 2x + C_1 \Rightarrow f(x) = y = C e^{2x}$
确定常数 $C$：
取 $x = 0$ 代入原积分方程：$f(0) = \int_0^0 \dots - \ln 2 = -\ln 2$
因此 $C = -\ln 2$。
最终结果：$f(x) = -\ln 2 \cdot e^{2x}$

### 💡 AI 的总结与思考

这份笔记展示了处理微分方程的第一种标准技术：**“化繁为简，分而治之”**。

#### 1. 核心数学思想：解耦（Decoupling）
微分方程难处理是因为 $x$ 和 $y$ 混合在一起。可分离变量法的本质就是通过代数手段让 $x$ 和 $y$ “分家”，从而把“微分方程问题”降级为“积分计算问题”。

#### 2. 注意细节：漏解风险
在例 1 中，你在分母除以 $y$ 之前先讨论了 $y=0$ 的情况。**这是一个非常专业的习惯**。在保研面试或高数竞赛中，直接除以 $y$ 而不讨论 $y=0$ 往往会被扣掉“完备性”分。

#### 3. 跨领域关联：补例 4 的深刻含义
补例 4 实际上是一个**变限积分方程**。这种形式在控制理论和概率论（如马尔可夫链的平稳分布推导）中经常出现。你通过求导将其转化为微分方程的思路非常清晰，这是处理积分方程的通用“套路”。
