## 微分算子法

**应用：** 线性非齐次微分方程特解 $y'' + py' + qy = f(x)$。

### 一、 算法引入

记 $D$ 为微分算子，定义运算：

1.  $D \cdot y = y', \quad D^2 y = y'', \quad D^n y = y^{(n)}$
2.  $\frac{1}{D} \cdot y = \int y dx, \quad \frac{1}{D^2} y = \iint y dx \cdot dx$ <span style="color:blue">(不加C)</span>

### 二、 特解一般形式

$y'' + py' + qy = f(x) \Rightarrow y^* = \frac{1}{D^2 + pD + q} f(x)$。

### 三、 性质

*   **线性：** $D(ay_1 + by_2) = aDy_1 + bDy_2$
*   **交换律：** $D^m \cdot D^n = D^n \cdot D^m = D^{m+n}$
*   **指数函数特性：** $P(D)e^{kx} = P(k)e^{kx}$
*   **三角函数特性：** $P(D^2)\sin(kx) = P(-k^2)\sin(kx)$ <span style="color:blue">($\cos kx$同)</span>。
*   **平移公式：** <span style="color:gray">[注：此处原文写法]</span> $e^{ax}P(D)y = P(D-a)(e^{ax}y)$
* **乘积求导：** $D^n(e^{kx}y) = e^{kx}(D+k)^n y$

---

### 四、 $f(x) = A \cdot e^{kx}$

**例1. $y'' + y' + y = e^x$**
$$y^* = \frac{1}{D^2 + D + 1} \cdot e^x = \frac{1}{3}e^x$$

**例2. $y'' - 3y' + 2y = e^{2x}$**
$$y^* = \frac{1}{D^2 - 3D + 2}e^{2x} \quad \text{\textcolor{blue}{(发现 $P(2)=0$)}}$$
问题原因：$(r-1)(r-2)=0 \Rightarrow k=2$。$\therefore y^* = x \cdot R_m(x)e^{2x}$，形式就不对了。

---

**修正：** 若 $e^{\lambda x}$ 中 $\lambda$ 为 $P(D)$ 的 $k$ 重特征根时，<span style="color:blue">(标准步骤展开处理)</span> **重根修正公式**。

$$P(D) = (D-k)^m \cdot Q(D), \quad (Q(k) \neq 0)$$
则 $y^* = \frac{1}{(D-k)^m Q(D)} e^{\lambda x}$

**推导：**
$(D-k)y_1 = \frac{e^{kx}}{Q(k)} \Rightarrow y_1' e^{-kx} - k e^{-kx} y_1 = \frac{1}{Q(k)} = d(y_1 e^{-kx}) \Rightarrow y_1 = \frac{xe^{kx}}{Q(k)}$
$(D-k)y_2 = y_1 = \frac{xe^{kx}}{Q(k)} \Rightarrow y_2' e^{-kx} - k e^{-kx} y_2 = \frac{x}{Q(k)} = d(y_2 e^{-kx}) \Rightarrow y_2 = \frac{x^2 e^{kx}}{2! Q(k)}$
$(D-k)y_3 = y_2 = \frac{x^2 e^{kx}}{2! Q(k)} \Rightarrow y_3' e^{-kx} - k e^{-kx} y_3 = \frac{x^2}{2! Q(k)} = d(y_3 e^{-kx}) \Rightarrow y_3 = \frac{x^3 e^{kx}}{3! Q(k)}$

$\therefore \frac{1}{(D-k)^m} y^* = \frac{1 \cdot e^{\lambda x}}{Q(\lambda)} \Rightarrow y^* = \frac{1 \cdot x^m e^{\lambda x}}{m! Q(\lambda)}$。
$\because P(D) = (D-\lambda)^m Q(D), \therefore P^{(m)}(D) = m! Q(D) + (D-\lambda) = m! Q(D)$。
$\therefore y^* = \frac{1 \cdot x^m e^{\lambda x}}{P^{(m)}(\lambda)}$。

**续例2：$y'' - 3y' + 2y = e^{2x}$，其中 $m=1, \lambda=2$。**
$$\therefore y^* = \frac{x e^{2x}}{(D^2 - 3D + 2)'|_{D=2}} = \frac{x e^{2x}}{1} = x e^{2x}$$

**例3. $y'' + 2y' + y = 2e^{-x}$**
$$(r+1)^2 = 0 \Rightarrow r_{1,2} = -1 \quad \therefore y^* = \frac{2 \cdot x^2 e^{-x}}{(r^2+2r+1)''|_{r=-1}} = \frac{2x^2 e^{-x}}{2} = x^2 e^{-x}$$

### 五、 $f(x) = A\sin \beta x + B\cos \beta x$

**例1. $y'' - y = \cos x$**
$$y^* = \frac{1}{D^2 - 1} \cdot \cos x = \frac{1}{-1-1}\cos x = -\frac{1}{2}\cos x$$

**例2. $y'' + y' + y = \cos x$**
$$y^* = \frac{1}{D^2 + D + 1} \cos x = \frac{1}{D} \cos x = \sin x$$

**例3. $y'' + y' + y = \sin 2x$**
$$y^* = \frac{1}{D^2 +D + 1} \sin 2x = \frac{1}{D-3} \sin 2x = \frac{D+3}{D^2-9} \sin 2x = \frac{D+3}{-13} \sin 2x = -\frac{2}{13}\cos 2x - \frac{3}{13}\sin 2x$$
<span style="color:blue">和留数定理有异曲同工之妙。</span>

**例4. $y'' + 4y = \cos 2x$**
$$y^* = \frac{1}{D^2 + 4} \cos 2x \quad \text{重根修正之术：} \quad e^{i\theta} = \cos \theta + i\sin \theta$$
$\begin{cases} \cos \omega x = \text{Re}(e^{i\omega x}) \\ \sin \omega x = \text{Im}(e^{i\omega x}) \end{cases}$
$\therefore \text{RHS} = \frac{1}{P(D)} \text{Re}/\text{Im}(e^{i\omega x}) = \text{Re}/\text{Im}(\frac{1}{P(D)}e^{i\omega x})$。

**由指数函数的推导修正公式知：** $\frac{1}{P(D)} e^{i\omega x} = \frac{x^m}{m! P^{(m)}(i\omega)} e^{i\omega x}$
$\because P(D) = (D^2+k^2)Q(D), P^{(m)}(ik) = m!(2ik)^m Q(-k^2)$。现针对 $m=1$ 的情况。

则 $\frac{1}{P(D)} e^{i\omega x} = \frac{x^m (\cos \omega x + i\sin \omega x)}{2i\omega Q(-k^2)} = \frac{x^m (i\cos \omega x - \sin \omega x)}{2\omega Q(-k^2)}$。
$\Rightarrow$ 仅用于判断外乘 $\cos \omega x / \sin \omega x$。

故 $\frac{1}{P(D)} \cos \omega x = \frac{x^m}{(m!) \text{Im}(P^{(m)}(ik))} \sin \omega x$。
$\frac{1}{P(D)} \sin \omega x = \frac{x^m}{(m!) \text{Re}(P^{(m)}(ik))} \cos \omega x$。
<span style="color:blue">同理，过程解的特征值：先提一个x，分母再导。</span>

**续例4. $y^* = \frac{1}{D^2+4}\cos 2x = \frac{x}{2D}\cos 2x = \frac{x}{4}\sin 2x$**

### 六、 $f(x) = a_0 + a_1 x + \dots + a_n x^n = P_n(x)$

**例1. $y'' + y = x + 1$**
$$y^* = \frac{1}{D^2+1}(x+1)$$
微分算子泰勒展开：$\begin{cases} \frac{1}{1-D} = 1 + D + D^2 + \dots D^n (\text{无穷级数等比}) \\ \frac{1}{1+D} = 1 - D + D^2 - \dots \end{cases}$ ($D$为无穷小量)。

$$\therefore y^* = (1 - D^2 + O(D^3))(x+1) = x+1$$

**例2. $y'' + y' + y = x + 1$**
$$y^* = \frac{1}{1+(D+D^2)}(x+1) = [1 - (D+D^2)](x+1) = x$$

**例3. $y'' + y' + y = x^2 + 1$**
$$y^* = \frac{1}{1+D+D^2}(x^2+1) = [1 - (D+D^2) + (D+D^2)^2](x^2+1) = x^2 + 1 - (2x+2) + 2 = x^2 - 2x + 1$$

**例4. $y'' - y' = x$**
$$y^* = \frac{1}{-D+D^2} \cdot x = \frac{1}{-D(1-D)} x = -\frac{1}{D}(1+D)x = -\frac{1}{D}(x+1) = -(\frac{1}{2}x^2 + x)$$

### 七、 $f(x) = A e^{kx} \sin \beta x / A e^{kx} P_n(x)$

**利用平移公式：** $y^* = \frac{1}{P(D)} A e^{kx} \sin \beta x / P_n(x) = A e^{kx} \frac{1}{P(D+k)} \sin \beta x / P_n(x)$

**例1. $y'' - 2y' = 2e^x \sin 2x$**
$$y^* = \frac{1}{D(D-2)} (2e^x \sin 2x) = 2e^x \frac{1}{(D+1)(D-1)} \sin 2x = 2e^x \frac{1}{D^2-1} \sin 2x = -\frac{2}{5} e^x \sin 2x$$

**例2. $y'' - y = e^x \cos 2x$**
$$y^* = \frac{1}{D^2-1} e^x \cos 2x = e^x \frac{1}{(D+1)^2-1} \cos 2x = e^x \frac{1}{D^2+2D} \cos 2x = e^x \frac{1}{2D-4} \cos 2x$$
$$= \frac{1}{2} e^x \frac{D+2}{D^2-4} \cos 2x = \frac{1}{2} e^x \frac{D+2}{-8} \cos 2x = -\frac{1}{16} e^x (\cos 2x - \sin 2x)$$

**例3. $y'' + y' + y = e^{-x} \cos x$**
$$y^* = \frac{1}{D^2+D+1} e^{-x} \cos x = e^{-x} \frac{1}{(D-1)^2+(D-1)+1} \cos x = e^{-x} \frac{1}{D^2-D+1} \cos x = e^{-x} \frac{1}{-D} \cos x = -e^{-x} \sin x$$

**例4. $y'' - 2y' + 10y = e^x \sin 3x$** ($D^2-2D+10$)
$$y^* = \frac{1}{(D-1)^2+9} e^x \sin 3x = e^x \frac{1}{D^2+9} \sin 3x = e^x \cdot \frac{x}{2D} \sin 3x = \frac{1}{6} x e^x \cos 3x$$

**例5. $y'' + y' + y = (x+1)e^x$**
$$y^* = \frac{1}{D^2+D+1}(x+1)e^x = e^x \frac{1}{(D+1)^2+(D+1)+1}(x+1) = \frac{1}{3} e^x \frac{1}{1+(D+\frac{1}{3}D^2)}(x+1)$$
$$= \frac{1}{3} e^x (1 - D - \frac{1}{3}D^2)(x+1) = \frac{1}{3} e^x (x)$$

**例6. $y'' + 3y' + 2y = 3xe^{-x}$**
$$y^* = \frac{1}{D^2+3D+2} 3xe^{-x} = e^{-x} \cdot \frac{1}{(D-1)^2+3(D-1)+2} \cdot 3x = 3e^{-x} \frac{1}{D(1+D)} x = 3e^{-x} \cdot \frac{1}{D}(1-D) x$$
$$= 3e^{-x} \cdot \frac{1}{D}(x-1) = 3e^{-x}(\frac{1}{2}x^2 - x)$$

**例7. $y'' - 6y' + 9y = (x+1)e^{3x}$**
$$y^* = \frac{1}{(D-3)^2}(x+1)e^{3x} = e^{3x} \cdot \frac{1}{D^2}(x+1) = e^{3x}(\frac{1}{6}x^3 + \frac{1}{2}x^2)$$

**例8. $y'' + 2y' + y = x^2 e^{x}$**
$$y^* = \frac{1}{(D+1)^2} x^2 e^{x} = e^{x} \frac{1}{D^2+4D+4} x^2 = \frac{1}{4}e^x \frac{1}{1+(D+\frac{1}{4}D^2)} x^2 = \frac{1}{4}e^x (1 - D - \frac{1}{4}D^2 + (D+\frac{1}{4}D^2)^2) x^2 = \frac{1}{4}e^x(x^2 - 2x + \frac{3}{2})$$
