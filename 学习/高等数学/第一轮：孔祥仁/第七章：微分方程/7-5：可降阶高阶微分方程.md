# 笔记标题：7-5 可降阶的高阶微分方程



---

## 一、 $y^{(n)} = f(x)$ 型
**核心思想：** 连续积分法。由于右端仅含自变量 $x$，通过 $n$ 次积分即可求得通解。

### 例 1：求解 $y''' = e^{2x} \cos x$
**求解过程：**
1. 第一次积分：
   $$y'' = \int e^{2x} \cos x \, dx = \frac{1}{2} e^{2x} \sin x + 2C_1$$
2. 第二次积分：
   $$y' = \int (\frac{1}{2} e^{2x} \sin x + 2C_1) \, dx = \frac{1}{4} e^{2x} \cos x + 2C_1 x + C_2$$
3. 第三次积分：
   $$y = \frac{1}{8} e^{2x} \sin x + C_1 x^2 + C_2 x + C_3$$

> **[个人总结]**：此类题目虽然简单，但计算量会随阶数增加。注意每一阶积分产生的任意常数 $C_i$ 必须保留， $n$ 阶方程通解中必须包含 $n$ 个独立任意常数。

## 二、 $y'' = f(x, y')$ 型（缺 $y$ 型）
**核心思想：** 通过换元降阶，将二阶方程转化为一阶方程。

**具体步骤：**
1. 令 $y' = p$，则 $y'' = p' = \frac{dp}{dx}$。
2. 代入原方程得：$p' = f(x, p)$。
3. 解出 $p = \phi(x, C_1)$。
4. 根据 $\frac{dy}{dx} = \phi(x, C_1)$，再次积分得：$y = \int \phi(x, C_1) \, dx + C_2$。

### 例题：求初值问题 $(x^2+1)y'' = 2xy', \quad y(0)=1, y'(0)=3$
**求解过程：**
1. 令 $y' = p, \quad y'' = p'$。
2. 原方程变为：$p' = \frac{2x}{x^2+1} \cdot p \Rightarrow \frac{dp}{p} = \frac{2x}{x^2+1} \, dx$。
3. 两边积分：$\ln p = \ln(x^2+1) + \ln C_1 \Rightarrow y' = C_1(x^2+1)$。
4. 代入初值 $y'(0)=3 \Rightarrow 3 = C_1(0+1) \Rightarrow C_1=3$。
   即：$y' = 3(x^2+1)$。
5. 再次积分：$y = x^3 + 3x + C_2$。
6. 代入初值 $y(0)=1 \Rightarrow 1 = 0+0+C_2 \Rightarrow C_2=1$。
7. **结果：** $y = x^3 + 3x + 1$。

### 补例：求通解 $xy'' + 3y' = 0$
1. 令 $y' = p, \quad y'' = p'$，则 $xp' = -3p$。
2. 分离变量：$\frac{dp}{p} = -3 \cdot \frac{dx}{x} \Rightarrow \ln p = -3 \ln x + \ln C_1 \Rightarrow y' = \frac{1}{x^3} C_1$。
3. 再次积分：$y = -\frac{1}{2x^2} C_1 + C_2$。
4. 令 $C_1' = -\frac{1}{2} C_1$，结果为：$y = \frac{C_1'}{x^2} + C_2$。

## 三、 $y'' = f(y, y')$ 型（缺 $x$ 型）
**核心思想：** 改变自变量，利用复合函数求导法则。

**具体步骤：**
1. 令 $y' = p$，则 $y'' = \frac{dp}{dx} = \frac{dp}{dy} \cdot \frac{dy}{dx} = p \frac{dp}{dy}$。
2. 方程变为：$p \frac{dp}{dy} = f(y, p)$（变为关于 $y$ 的一阶方程）。
3. 解得 $p = \phi(y, C_1)$，即 $\frac{dy}{dx} = \phi(y, C_1)$。
4. 分离变量：$dx = \frac{1}{\phi(y, C_1)} \, dy \Rightarrow \int \frac{1}{\phi(y, C_1)} \, dy = x + C_2$。

### 例 1：$y \cdot y'' - y'^2 = 0$
1. 令 $y'=p, y'' = p \frac{dp}{dy}$。
2. $y \cdot p \frac{dp}{dy} = p^2 \Rightarrow \frac{dp}{p} = \frac{dy}{y}$。
3. 积分得：$p = C_1 y \Rightarrow \frac{dy}{dx} = C_1 y$。
4. 分离变量：$\frac{dy}{y} = C_1 dx \Rightarrow \ln y = C_1 x + \ln C_2$。
5. **结果：** $y = e^{C_1 x} \cdot C_2$。

### 补例 2：通解 $y'' + \frac{y'^2}{1-y} = 0$
1. 令 $y'=p, y'' = p \frac{dp}{dy}$。
2. $p \frac{dp}{dy} = \frac{p^2}{y-1} \Rightarrow \frac{dp}{p} = \frac{dy}{y-1}$。
3. 积分得：$p = C_1(y-1) \Rightarrow \frac{dy}{dx} = C_1(y-1)$。
4. 分离变量：$\frac{dy}{y-1} = C_1 dx \Rightarrow \ln(y-1) = C_1 x + \ln C_2$。
5. **结果：** $y = C_2 e^{C_1 x} + 1$ （注意：此处 $C_2 \neq 0$）。

### 补例 3：初值问题 $yy'' + y'^2 = 0, \quad y(0)=1, y'(0)=\frac{1}{2}$
1. 令 $y'=p, y'' = p \frac{dp}{dy}$。
2. $y \cdot p \frac{dp}{dy} = -p^2 \Rightarrow \frac{dp}{p} = -\frac{dy}{y}$。
3. 积分得：$p = \frac{C_1}{y}$。
4. 代入 $y=1, p=\frac{1}{2} \Rightarrow C_1 = \frac{1}{2} \Rightarrow y' = \frac{1}{2y}$。
5. 还原：$2y \cdot dy = dx \Rightarrow y^2 = x + C_2$。
6. 代入 $y(0)=1 \Rightarrow 1^2 = 0 + C_2 \Rightarrow C_2 = 1$。
7. **结果：** $y^2 = x + 1$。

## [深度思考与笔记总结]

### 1. 降阶法的逻辑链条
降阶法的本质是**通过变量代换减少独立变量的数量**。
*   **缺 $y$：** 既然没有 $y$，我们就把 $y'$ 提升为主角，直接建立从 $y'$ 到 $x$ 的桥梁。
*   **缺 $x$：** 既然没有 $x$，我们就通过链式法则 $\frac{dp}{dx} = p \frac{dp}{dy}$，强行将 $y$ 变成自变量。

### 2. 注意事项（易错点）
*   **常数的处理：** 在处理 $y^{(n)} = f(x)$ 时，每一层积分都要加常数，不能最后统一加，否则会漏掉项（例如漏掉 $C_1 x$）。
*   **分母不为零的讨论：** 笔记中特别标注了 $C_2 \neq 0$，这体现了严谨性。在除以 $p$ 或 $y-1$ 时，应当考虑这些值等于 $0$ 是否为特解。
