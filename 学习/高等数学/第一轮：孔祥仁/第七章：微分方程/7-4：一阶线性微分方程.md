# 笔记：7-4 一阶线性微分方程

![封面图占位符：草地上拿着捕蝶网的小熊、企鹅和狐狸，文字为 SPRING](image_placeholder_page1.png)

## 一、形式
$$\frac{dy}{dx} + P(x)y = Q(x)$$

1. **$Q(x) = 0$**：称为一阶齐次线性微分方程。
2. **$Q(x) \neq 0$**：称为一阶非齐次线性微分方程。

---

## 二、解法

### 1. 齐次形式解
$\frac{dy}{dx} = -P(x)y \Rightarrow \frac{1}{y}dy = -P(x)dx$
积分得：**$y = C e^{-\int P(x)dx}$**

### 2. 非齐次形式解（常数变易法）
设 **$y = u(x) e^{-\int P(x)dx}$** 是非齐次的解。
对 $y$ 求导：
$y' = u'(x) e^{-\int P(x)dx} + u(x) e^{-\int P(x)dx} \cdot [-P(x)]$
代入原方程：
$e^{-\int P(x)dx} [u'(x) - u(x)P(x)] + P(x) \cdot u(x) e^{-\int P(x)dx} = Q(x)$
化简得：$u'(x) e^{-\int P(x)dx} = Q(x)$
$\Rightarrow u'(x) = Q(x) e^{\int P(x)dx}$
$\Rightarrow u(x) = \int Q(x) e^{\int P(x)dx} dx + C$

**通解公式**：
$$y = e^{-\int P(x)dx} \left[ \int Q(x) e^{\int P(x)dx} dx + C \right]$$
> **注（蓝色手写）**：这个公式反映了：非齐通 = 齐通 + 非齐特解。

## 三、例题提取

### 例 1：求 $\frac{dy}{dx} - \frac{2y}{x+1} = (x+1)^{\frac{5}{2}}$ 的通解

- **齐次部分**：$\frac{dy}{dx} = \frac{2y}{x+1} \Rightarrow \frac{1}{y}dy = \frac{2}{x+1}dx \Rightarrow \ln y = 2\ln(x+1) + \ln C$
  $\Rightarrow y = C(x+1)^2$
- **常数变易**：令 $y^* = u(x)(x+1)^2$
  代入得：$u'(x)(x+1)^2 = (x+1)^{\frac{5}{2}}$
  $\Rightarrow u'(x) = (x+1)^{\frac{1}{2}} \Rightarrow u(x) = \frac{2}{3}(x+1)^{\frac{3}{2}}$
- **特解**：$y^* = \frac{2}{3}(x+1)^{\frac{3}{2}}(x+1)^2 = \frac{2}{3}(x+1)^{\frac{7}{2}}$
- **结果**：$y = C(x+1)^2 + \frac{2}{3}(x+1)^{\frac{7}{2}}$

### 补例 1：求通解 $x y' \ln x + y = ax(1 + \ln x)$

- **标准型**：$\frac{dy}{dx} + \frac{y}{x \ln x} = \frac{ax(1+\ln x)}{x \ln x} = a + \frac{a}{\ln x}$
- **计算积分项**：
  $-\int P(x)dx = -\int \frac{dx}{x \ln x} = -\ln(\ln x) \Rightarrow e^{-\int P(x)dx} = \frac{1}{\ln x}$
  $\int Q(x) e^{\int P(x)dx} dx = \int (a + \frac{a}{\ln x}) \ln x dx = \int (a \ln x + a) dx = a(x \ln x)$
- **代入公式**：$y = \frac{1}{\ln x} [ax \ln x + C]$
- **结果**：$y = ax + \frac{C}{\ln x}$

---

### 补例 2：求通解 $(x - 2xy - y^2) \frac{dy}{dx} + y^2 = 0$

- **反转变量**：将 $x$ 看作 $y$ 的函数。
  $\frac{dx}{dy} = \frac{2xy + y^2 - x}{y^2} = \frac{2}{y}x + 1 - \frac{x}{y^2}$
  $\Rightarrow \frac{dx}{dy} + (\frac{1}{y^2} - \frac{2}{y})x = 1$
- **计算**：$P(y) = \frac{1}{y^2} - \frac{2}{y}, Q(y) = 1$
  $e^{-\int P(y)dy} = e^{\frac{1}{y} + 2\ln y} = y^2 e^{\frac{1}{y}}$
  $\int Q(y) e^{\int P(y)dy} dy = \int 1 \cdot e^{-\frac{1}{y} - 2\ln y} dy = \int e^{-\frac{1}{y}} \cdot \frac{1}{y^2} dy = e^{-\frac{1}{y}} + C$
- **结果**：$x = y^2 e^{\frac{1}{y}} [e^{-\frac{1}{y}} + C] = y^2 + C y^2 e^{\frac{1}{y}}$

---

### 补例 3：通过积分方程求函数 $f(x) = \int_0^{3x} f(\frac{t}{3})dt + e^{2x}$

- **求导**：$f'(x) = f(x) \cdot 3 + 2e^{2x} \Rightarrow y' - 3y = 2e^{2x}$
- **公式法**：$P(x) = -3, Q(x) = 2e^{2x}$
  $y = e^{3x} [\int 2e^{2x} e^{-3x} dx + C] = e^{3x} [-2e^{-x} + C]$
  $\Rightarrow y = -2e^{2x} + C e^{3x}$
- **初值**：取 $x=0, f(0) = 0 + e^0 = 1$
  $1 = -2 + C \Rightarrow C = 3$
- **结果**：$f(x) = -2e^{2x} + 3e^{3x}$

## 四、伯努利方程 (Bernoulli Equation)
> **目标**：转换为线性微分方程。

1. **形式**：$\frac{dy}{dx} + P(x)y = Q(x)y^n \quad (n \neq 0, 1)$
2. **解法**：
   两边同除以 $y^n \Rightarrow y^{-n} \frac{dy}{dx} + P(x) y^{1-n} = Q(x)$
   令 $z = y^{1-n}$，则 $\frac{dz}{dx} = (1-n) y^{-n} \frac{dy}{dx}$
   代入得：**$\frac{dz}{dx} + (1-n) P(x) z = (1-n) Q(x)$**

---

### 例 1 (伯努利)：$\frac{dy}{dx} + \frac{y}{x} = a \ln x \cdot y^2$

- **变换**：除以 $y^2 \Rightarrow y^{-2} y' + \frac{1}{x} y^{-1} = a \ln x$
- **令 $z = y^{-1}$**，则 $z' = -y^{-2} y' \Rightarrow -z' + \frac{1}{x}z = a \ln x \Rightarrow z' - \frac{1}{x}z = -a \ln x$
- **公式求解**：$z = e^{\ln x} [\int -a \ln x \cdot \frac{1}{x} dx + C] = x [-\frac{a}{2} \ln^2 x + C]$
- **还原**：$y = \frac{1}{x [C - \frac{a}{2} \ln^2 x]}$

### 补例 1. 求通解 $2xy^3 y' + x^4 - y^4 = 0$

**解：**
$$\frac{dy}{dx} - \frac{y}{2x} = -\frac{1}{2}x^3 y^{-3}$$
$$y^3 \frac{dy}{dx} - \frac{y^4}{2x} = -\frac{x^3}{2}$$

令 $z = y^4$，则 $\frac{dz}{dx} = 4y^3 \frac{dy}{dx}$
$$\therefore \frac{dz}{dx} - \frac{2}{x} \cdot z = -2x^3$$

此处 $P(x) = -\frac{2}{x}, Q(x) = -2x^3$
通解公式：$z = e^{-\int P(x)dx} \left[ \int Q(x) e^{\int P(x)dx} dx + C \right]$

计算积分项：
$$\int P(x)dx = -2 \ln x$$
$$\int Q(x) e^{\int P(x)dx} dx = \int -2x^3 \cdot \frac{1}{x^2} dx = \int -2x dx = -x^2$$

代入公式：
$$\therefore z = x^2 [-x^2 + C] = -x^4 + Cx^2 = y^4$$

### 💡 AI 的总结与思考

在这部分笔记中，你已经完成了从“基础一阶线性”到“复杂非线性（伯努利）”的进阶。

1.  **常数变易法的本质**：它是一种极其高明的“猜解”策略。既然齐次方程的解长那个样子，我们就假设非齐次的解只是把常数 $C$ 变成了动态的函数 $u(x)$。
2.  **伯努利方程的降维打击**：伯努利方程虽然是非线性的，但通过 $z = y^{1-n}$ 的代换，它被完美“掰直”成了线性方程。这种**化归思想**是解决所有高级微分方程的通法。
3.  **变量转换的灵活性**：补例 2（关于 $x$ 求导）提醒我们，有时候微分方程的“自变量”和“因变量”是可以互换位置的。如果发现关于 $y$ 的方程太难，不妨看看关于 $x$ 的方程是否是线性的。
