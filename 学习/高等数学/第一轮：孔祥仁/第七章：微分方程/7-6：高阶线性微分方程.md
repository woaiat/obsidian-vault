## 7-6 高阶线性微分方程

### 一、 线性相关与线性无关 (Linear Dependence / Independence)

#### 1. 线性相关 (Linear Dependence)
对于函数组 $y_1(x), y_2(x), \dots, y_n(x)$，如果存在一组**不全为 0** 的常数 $k_1, k_2, \dots, k_n$，使得恒等式：
$$k_1 y_1 + k_2 y_2 + \dots + k_n y_n = 0$$
成立，则称该函数组在给定区间内是**线性相关**的。

*   **推导性质：**
    若设 $k_n \neq 0$，则可以将 $y_n$ 表示为其他函数的线性组合：
    $$y_n = -\frac{1}{k_n} [k_1 y_1 + k_2 y_2 + \dots + k_{n-1} y_{n-1}]$$
*   **特殊情况 ($n=2$)：**
    若 $k_1 y_1 + k_2 y_2 = 0$，则有 $\frac{y_1}{y_2} = -\frac{k_2}{k_1} = C$（常数）。即：**两个函数线性相关的本质是其比值为常数。**

#### 2. 线性无关 (Linear Independence)
如果不存在不全为 0 的常数使得上述恒等式成立（即只有当 $k_1 = k_2 = \dots = k_n = 0$ 时等式才成立），则称该函数组为**线性无关**（没有多余项）。

*   **特殊情况 ($n=2$)：**
    若 $\frac{y_1}{y_2} \neq C$（比值不为常数），则 $y_1, y_2$ 线性无关。

### 二、 高阶线性微分方程解的结构

#### 1. 方程形式
考虑二阶线性微分方程：
$$y'' + P(x)y' + Q(x)y = f(x)$$

*   **① 齐次形式：** 若 $f(x) \equiv 0$，称为**二阶齐次线性微分方程**。
*   **② 非齐次形式：** 若 $f(x) \neq 0$，称为**二阶非齐次线性微分方程**。

#### 2. 齐次方程的解结构：$y'' + P(x)y' + Q(x)y = 0$ —— (A)
*   **定理 1 (叠加原理)：** 若 $y_1(x), y_2(x)$ 是方程 (A) 的解，则其线性组合
    $$y = C_1 y_1(x) + C_2 y_2(x)$$
    也是方程 (A) 的解。（注意：但不一定是通解）
*   **定理 2 (通解结构)：** 若 $y_1(x)$ 和 $y_2(x)$ 是方程 (A) 的两个**线性无关**的解，则
    $$y = C_1 y_1(x) + C_2 y_2(x)$$
    是该方程的**通解**。
*   **③ 推论 (n阶扩展)：** 若 $y_1(x), \dots, y_n(x)$ 为 $n$ 阶齐次线性微分方程的 $n$ 个线性无关解，则其通解为：
    $$y = C_1 y_1 + \dots + C_n y_n$$

#### 3. 非齐次方程的解结构：$y'' + P(x)y' + Q(x)y = f(x) \neq 0$ —— (B)
*   **① 定理 3：** 设 $y^*$ 是非齐次方程 (B) 的一个**特解**，$Y(x)$ 是对应齐次方程 (A) 的**通解**。则
    $$y(x) = Y(x) + y^*$$
    是非齐次方程 (B) 的**通解**。
    > **公式总结：** 非齐次通解 = 对应齐次通解 + 非齐次特解。
*   **② 推论：** 两个非齐次方程的解之差（非齐特1 - 非齐特2）是对应齐次方程的解。

*   **③ 定理 4 (叠加原理)：**
    针对复杂的项 $f(x) = f_1(x) + f_2(x)$，考虑方程 (C)：
    $$y'' + P(x)y' + Q(x)y = f_1(x) + f_2(x)$$
    若 $y_1(x)$ 是 $y'' + P(x)y' + Q(x)y = f_1(x)$ 的解，
    且 $y_2(x)$ 是 $y'' + P(x)y' + Q(x)y = f_2(x)$ 的解，
    则 $y = y_1(x) + y_2(x)$ 是原方程 (C) 的特解。

## 个人总结与思考 (Summary & Reflection)

### 1. 核心逻辑链条
这部分笔记展示了线性微分方程解的高度**对称性**与**线性特征**。学习的路径如下：
1.  **定义判别：** 通过比值（针对二阶）或线性组合判定函数是否“多余”。
2.  **由简入繁：** 先研究 $f(x)=0$ 的齐次情况，利用线性无关解构造通解池。
3.  **拼图原理：** 非齐次方程的解就像是在齐次通解（背景）上叠加一个特解（特定的偏移）。

### 2. 重点提醒
*   **线性无关的判定：** 虽然笔记中提到了比值法（适用于二阶），但在更高阶中，通常需要借助 **Wronskian (朗斯基) 行列式** 来判定。
*   **叠加原理的妙用：** 在处理复杂的非齐次项（如 $e^x + \sin x$）时，定理 4 允许我们拆分求解，这大大降低了待定系数法的计算难度。
