## 一、映射
$f: X \to Y$，$f$ 称为从 $X$ 到 $Y$ 的**映射**。
- **定义域 ($D_f$)**：$D_f = X$
- **值域 ($R_f$)**：$R_f = f(X) \subset Y$
- **其中**：$X, Y$ 是非空集合。

### 三要素：$X, f, R_f$
1. $X = D_f, R_f \subset Y$。
2. **唯一性**：对于每一个 $x \in X$，有且仅有一个 $y \in Y$ 与之对应。
   - 注意：可以“多对一”，不可以“一对多”。
   - 当 $R_f = Y$ 时，称之为**满射**。
   - 当 $x_1 \neq x_2 \Rightarrow f(x_1) \neq f(x_2)$ 时，称之为**单射**。
   - 既是满射又是单射时，称为**一一映射（双射）**。

### 逆映射与复合映射
- **逆映射**：设 $f: X \to Y$ 是双射，则对于每个 $y \in Y$，存在唯一的 $x \in X$ 满足 $f(x) = y$。
  - 记为：$g = R_f \to X$，$f^{-1}$。$D_{f^{-1}} = R_f, R_{f^{-1}} = X$。
- **复合映射**：$g: X \to Y_1$，$f: Y_2 \to Z$。且 $R_g \subset Y_2, x \in X$。则：
  - $f[g(x)] \in Z$，写为 $f \circ g: X \to Z$。**条件**：$R_g \subset D_f$。

## 二、函数：自变量和因变量的对应
$f: D \to \mathbb{R}, y = f(x)$（实变数集）。
- **两要素**：$D_f, f$。
- **符号函数**：$y = \text{sgn } x = \begin{cases} -1 & x < 0 \\ 0 & x = 0 \\ 1 & x > 0 \end{cases}$
  - 关系式：$x = \text{sgn } x \cdot |x|$
- **取整函数**：$[x]$，例如 $[2.7] = 2$。

### 特性
1. **有界性**：有上界且有下界 $\Leftrightarrow$ 有界。
   - 无界：$\forall M > 0, \exists x_1 \in X, |f(x_1)| > M$。
2. **单调性**。
3. **奇偶性**。
4. **周期性**：$\exists l > 0, f(x+l) = f(x)$。
   - **狄利克雷函数**：$D(x) = \begin{cases} 1, & x \in \mathbb{Q} \\ 0, & x \in \mathbb{Q}^C \end{cases}$
     - 它是周期函数，但**无最小正周期**（任何正有理数都是其周期）。
   - **注**：并非每个周期函数都有最小正周期。

### 反函数
- 设 $f: D \to f(D)$ 映射，则 $f^{-1}: f(D) \to D$。
- **性质**：若 $f$ 单调增加，则其反函数 $f^{-1}$ 也单调增加（单调性相同）。

# 笔记：函数运算与初等函数

## 函数运算
设 $f(x), g(x), D_f \cap D_g = D \neq \varnothing$：
- $(f \pm g)(x) = f(x) \pm g(x)$
- $(f \cdot g)(x) = f(x) \cdot g(x)$
- $(f/g)(x) = f(x)/g(x) \quad (g(x) \neq 0)$

### 奇偶性分解
任何一个定义在对称区间上的函数 $f(x)$ 都可以分解为一个奇函数 $h(x)$ 和一个偶函数 $g(x)$ 之和：
- $f(x) = g(x) + h(x)$
- $g(x) = \frac{1}{2}[f(x) + f(-x)]$（偶部分）
- $h(x) = \frac{1}{2}[f(x) - f(-x)]$（奇部分）

## 初等函数
### 幂、指、对、三、反三
- **反三角函数图像及定义域限制**：
  - $y = \arcsin x$：$[-\frac{\pi}{2}, \frac{\pi}{2}]$
  - $y = \arccos x$：$[0, \pi]$
  - $y = \arctan x$：$(-\frac{\pi}{2}, \frac{\pi}{2})$
  - $y = \text{arccot } x$：$(0, \pi)$
  - **重要恒等式**：$\arcsin x + \arccos x = \frac{\pi}{2}$

### 双曲函数
- **双曲正弦**：$\sinh x = \frac{1}{2}(e^x - e^{-x})$
- **双曲余弦**：$\cosh x = \frac{1}{2}(e^x + e^{-x})$
- **双曲正切**：$\tanh x = \frac{\sinh x}{\cosh x}$

**常用公式**：
- $\sinh(x \pm y) = \sinh x \cosh y \pm \cosh x \sinh y$
- $\cosh(x \pm y) = \cosh x \cosh y \pm \sinh x \sinh y$
- $\cosh^2 x - \sinh^2 x = 1$

### 其他三角函数
- **正割**：$\sec x = \frac{1}{\cos x}$
- **余割**：$\csc x = \frac{1}{\sin x}$

# 笔记：三角恒等式汇总

## 和差化积
*推导思路：取 $x = \frac{m+n}{2}, y = \frac{m-n}{2}$*
- $\sin x + \sin y = 2 \sin \frac{x+y}{2} \cos \frac{x-y}{2}$
- $\sin x - \sin y = 2 \cos \frac{x+y}{2} \sin \frac{x-y}{2}$
- $\cos x + \cos y = 2 \cos \frac{x+y}{2} \cos \frac{x-y}{2}$
- $\cos x - \cos y = -2 \sin \frac{x+y}{2} \sin \frac{x-y}{2}$

## 积化和差
- $\sin x \cos y = \frac{1}{2}[\sin(x+y) + \sin(x-y)]$
- $\cos x \sin y = \frac{1}{2}[\sin(x+y) - \sin(x-y)]$
- $\cos x \cos y = \frac{1}{2}[\cos(x+y) + \cos(x-y)]$
- $\sin x \sin y = -\frac{1}{2}[\cos(x+y) - \cos(x-y)]$

## 半角公式与万能公式
- $\tan \frac{x}{2} = \frac{\sin x}{1 + \cos x} = \frac{1 - \cos x}{\sin x}$

**万能公式（用 $t = \tan \frac{x}{2}$ 表示）**：
- $\sin x = \frac{2t}{1 + t^2}$
- $\cos x = \frac{1 - t^2}{1 + t^2}$
- $\tan x = \frac{2t}{1 - t^2}$

## 反三角函数导数（预习补充）
- $(\arcsin x)' = \frac{1}{\sqrt{1-x^2}}$
- $(\arccos x)' = -\frac{1}{\sqrt{1-x^2}}$
- $(\arctan x)' = \frac{1}{1+x^2}$
- $(\text{arccot } x)' = -\frac{1}{1+x^2}$

### 💡 AI 的总结与思考

这份笔记是非常扎实的基础复习。对于计算机科学（尤其是图形学和算法）来说，以下几点需要重点关注：

1.  **映射的单、满、双射**：这是离散数学和现代密码学的基础。在理解哈希函数（Hash Function）时，你会发现完美的哈希就是一个双射映射。
2.  **狄利克雷函数**：这是一个经典的“病态函数”，在实变函数论中非常重要，也是理解“测度”概念的起点。它在计算机模拟中几乎无法直接绘制，因为它在任何区间内都是极度不连续的。
3.  **万能公式**：在做积分运算（尤其是三角有理函数的积分）时，万能公式是最后的救命稻草，它能把超越函数运算转化为代数多项式运算。
4.  **双曲函数**：在机器学习中，你常用的激活函数 `tanh` 就在这里。理解它的指数定义，有助于你理解神经网络中梯度消失的问题。
