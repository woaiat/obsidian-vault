# 笔记：1-3 函数的极限

## 一、定义

函数的极限可以根据自变量 $x$ 的趋向分为两种主要情况：$x \to \infty$ 和 $x \to x_0$。

### 1. 自变量趋向于无穷大时函数的极限 ($x \to \infty$)
- **$x \to +\infty$**：$\forall \epsilon > 0, \exists X > 0$，当 $x > X$ 时，总有 $|f(x) - A| < \epsilon \Rightarrow \lim_{x \to +\infty} f(x) = A$。
- **$x \to -\infty$**：$\forall \epsilon > 0, \exists X > 0$，当 $x < -X$ 时，总有 $|f(x) - A| < \epsilon \Rightarrow \lim_{x \to -\infty} f(x) = A$。
- **$x \to \infty$**：$\forall \epsilon > 0, \exists X > 0$，当 $|x| > X$ 时，总有 $|f(x) - A| < \epsilon \Rightarrow \lim_{x \to \infty} f(x) = A$。
  - **几何意义**：水平渐近线。若 $\lim_{x \to \infty} f(x) = A$，则直线 $y = A$ 是曲线 $y = f(x)$ 的水平渐近线。

### 2. 自变量趋向于有限值时函数的极限 ($x \to x_0$)
- **邻域概念**：
  - **全邻域**：$(x_0 - \delta, x_0 + \delta)$，记作 $U(x_0, \delta)$。
  - **去心邻域**：$(x_0 - \delta, x_0) \cup (x_0, x_0 + \delta)$，记作 $\mathring{U}(x_0, \delta)$，即 $\{x | 0 < |x - x_0| < \delta\}$。
- **定义 ($\epsilon-\delta$ 语言)**：
  $\forall \epsilon > 0, \exists \delta > 0$，当 $0 < |x - x_0| < \delta$ 时，总有 $|f(x) - A| < \epsilon \Rightarrow \lim_{x \to x_0} f(x) = A$。
- **充要条件（红色手写补充）**：
  $$\lim_{x \to x_0} f(x) = A \Leftrightarrow f(x_0^+) = f(x_0^-) = A$$
  （即左右极限均存在且相等）。

### 3. 例题证明
**例**：证明 $\lim_{x \to \infty} \frac{1}{x} = 0$。
**证**：$\forall \epsilon > 0$，欲使 $\left| \frac{1}{x} - 0 \right| < \epsilon$，即 $\frac{1}{|x|} < \epsilon$，只需 $|x| > \frac{1}{\epsilon}$。
取 $X = \frac{1}{\epsilon}$，则当 $|x| > X$ 时，恒有 $|\frac{1}{x} - 0| < \epsilon$。得证。

## 二、性质

### 1. 唯一性
若 $\lim_{x \to x_0} f(x)$ 存在，则极限唯一。

### 2. 局部有界性
若 $\lim_{x \to x_0} f(x) = A$，则 $\exists \delta > 0$，当 $0 < |x - x_0| < \delta$ 时，总有 $|f(x)| \le M$（$M$ 为某正数）。
- **注意**：这是“局部”有界，脱离了去心邻域则不一定有界。

### 3. 局部保号性
1. 若 $\lim_{x \to x_0} f(x) = A$，且 $A > 0$，则 $\exists \delta > 0$，当 $x \in \mathring{U}(x_0, \delta)$ 时，$f(x) > 0$。
2. 若 $A < 0$，同理可得 $f(x) < 0$。
3. **推论**：若在 $x_0$ 的某去心邻域内 $f(x) \ge 0$，且 $\lim_{x \to x_0} f(x) = A$，则 $A \ge 0$。

### 💡 AI 的总结与思考

这一页笔记完成了从“数列”到“函数”的跨越。对于后续学习，这里有几个关键点值得深挖：

1.  **$\epsilon-N$ vs $\epsilon-\delta$**：
    - 数列极限考察的是“足够远以后”的情况（$n > N$）。
    - 函数极限考察的是“足够靠近某点”的情况（$0 < |x - x_0| < \delta$）。
    - **思考**：为什么函数极限定义里必须是“去心”邻域？因为极限研究的是趋向过程，与函数在 $x_0$ 点本身是否有定义、定义为何值完全无关。

2.  **渐近线的安全意义**：
    - 在网络流量建模中，水平渐近线常用来描述系统在长时间运行后的稳定状态（Steady State）。如果流量函数不收敛，可能意味着系统存在震荡或 DDoS 攻击导致的异常增长。

3.  **局部保号性的威力**：
    - 它是证明很多中值定理的有力武器。在算法分析中，如果我们知道一个代价函数在最优点附近的极限大于 0，我们就能保证在最优点附近的一个小范围内，该算法始终是有开销的。
