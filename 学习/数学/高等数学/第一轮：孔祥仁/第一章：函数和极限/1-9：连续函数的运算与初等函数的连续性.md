# 笔记：1-9 连续函数的运算与初等函数的连续性

## 一、连续函数的运算法则

**定理 1**：若函数 $f(x)$ 和 $g(x)$ 在点 $x_0$ 处连续，则其和、差、积、商（分母不为 0）在 $x_0$ 处仍连续。

**定理 2**：单调、连续函数的反函数也是单调、连续的。

**定理 3**：若 $\lim_{x \to x_0} g(x) = u_0$，$f(u)$ 在 $u = u_0$ 处连续，则：
$$\lim_{x \to x_0} f[g(x)] = \lim_{u \to u_0} f(u) = f(u_0)$$
（即极限符号可以与连续函数符号交换位置）。

**定理 4**：若 $u = g(x)$ 在 $x_0$ 处连续，$y = f(u)$ 在 $u_0 = g(x_0)$ 处连续，则复合函数 $y = f[g(x)]$ 在 $x_0$ 处连续。

## 二、初等函数的连续性

1. **基本初等函数**在其定义域内是连续的。
2. **初等函数**（由基本初等函数经有限次代数运算和复合而成）在其**定义区间**内是连续的。

## 三、经典求极限例题

1. $\lim_{x \to 0} \frac{\log_a(1+x)}{x} = \lim_{x \to 0} \log_a (1+x)^{1/x} = \log_a e = \frac{1}{\ln a}$
   - **推论**：$\ln(1+x) \sim x$

2. $\lim_{x \to 0} \frac{a^x - 1}{x} = \ln a$
   - **证明技巧**：设 $a^x - 1 = t$，则 $x = \log_a(1+t)$。
   - 原式 $= \lim_{t \to 0} \frac{t}{\log_a(1+t)} = \ln a$。
   - **推论**：$e^x - 1 \sim x$

3. $\lim_{x \to 0} \frac{(1+x)^a - 1}{x} = a$
   - **证明技巧**：利用 $u^v = e^{v \ln u}$。
   - 原式 $= \lim_{x \to 0} \frac{e^{a \ln(1+x)} - 1}{a \ln(1+x)} \cdot \frac{a \ln(1+x)}{x} = 1 \cdot a = a$。

4. $\lim_{x \to 0} \frac{(2x+1)^{\frac{3}{\sin x}}}{x} \to$ 属于 $1^\infty$ 型，结果为 $e^6$。

## 五、常用等价无穷小与泰勒展开对照表 ($x \to 0$)

### 1. 常用等价无穷小
1. $\sin x \sim \arcsin x \sim \tan x \sim \arctan x \sim \ln(x + \sqrt{1+x^2}) \sim x$
2. $\ln(1+x) \sim x$
3. $\log_a(1+x) \sim \frac{x}{\ln a}$
4. $e^x - 1 \sim x$
5. $a^x - 1 \sim x \ln a$
6. $\sqrt[n]{1+x} - 1 \sim \frac{1}{n}x$
7. $(1+x)^a - 1 \sim ax$
8. $1 - \cos x \sim \frac{1}{2}x^2 \sim \sec x - 1$
9. $x - \sin x \sim \frac{1}{6}x^3$
10. $\arcsin x - x \sim \frac{1}{6}x^3$
11. $\tan x - x \sim \frac{1}{3}x^3$
12. $x - \arctan x \sim \frac{1}{3}x^3$
13. $\tan x - \sin x \sim \frac{1}{2}x^3$
14. $e^x - (1+x) \sim \frac{1}{2}x^2$

### 2. 泰勒展开（麦克劳林展开）
- $e^x = 1 + x + \frac{x^2}{2!} + \dots + \frac{x^n}{n!} + o(x^n)$
- $\ln(1+x) = x - \frac{x^2}{2} + \frac{x^3}{3} - \dots$
- $\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} - \dots$
- $\cos x = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \dots$

### 3. $x \to 0$ 阶数直观排序 (记忆链条)
$$\arctan x < \sin x < x < \arcsin x < \tan x$$
*(都在 $\frac{1}{6}x^3$ 或 $\frac{1}{3}x^3$ 的量级微调)*

## 六、补充重要极限
1. $\lim_{x \to +\infty} x^{1/x} = 1$
   - **证**：$\lim_{x \to \infty} x^{1/x} = \lim e^{\frac{\ln x}{x}}$。因为 $\frac{\ln x}{x} \to 0$，故 $e^0 = 1$。
2. $\lim_{x \to 1} \frac{x^m - 1}{x^n - 1} = \frac{m}{n}$

### 💡 AI 的总结与思考

这份笔记是整个第一章“极限与连续”的**满分攻略图**。

#### 1. 泰勒展开 vs 等价无穷小
你把这两者放在一起对比非常明智。
- **等价无穷小**：其实就是泰勒展开的**第一项**（一阶项）。
- **避坑指南**：在做加减法时，如果一阶项相抵消了，等价无穷小就会失效（变成 0 导致精度丢失），这时候必须请出“泰勒展开”来提供二阶、三阶项的信息。

#### 2. “记忆链条”的妙用
你笔记中画出的那个数轴（$\arctan < \sin < x < \arcsin < \tan$）极其有用。
- 这不仅能帮你快速解填空题，还能让你在做**无穷小阶数比较**的题目时，不需要求导就能一眼看出答案。

#### 3. 计算机科学中的连续性
在网络安全和机器学习中，我们追求的往往不仅仅是“连续”，而是“**利普希茨连续**”（Lipschitz Continuity）。这意味着函数的变化速度不仅是平滑的，而且是有上限的。这保证了当你的攻击样本（Adversarial Examples）被加入微小扰动时，分类器的输出不会发生剧烈跳变。
