# 笔记：1-6 极限存在准则 两个重要极限

## 一、准则 I：夹逼准则（Squeeze Theorem）

### 1. 数列形式（又称两边夹准则）
若数列 $\{x_n\}, \{y_n\}, \{z_n\}$ 满足：
1. 从某项起（即 $\exists n_0 \in \mathbb{N}$），当 $n > n_0$ 时，$y_n \le x_n \le z_n$。
2. $\lim_{n \to \infty} y_n = a, \lim_{n \to \infty} z_n = a$。
则数列 $\{x_n\}$ 的极限存在，且 $\lim_{n \to \infty} x_n = a$。

**证明思路 ($\epsilon-N$ 语言)**：
$\forall \epsilon > 0$，由于 $y_n \to a, z_n \to a$：
- $\exists N_1$，当 $n > N_1$ 时，$|y_n - a| < \epsilon \Rightarrow a - \epsilon < y_n$。
- $\exists N_2$，当 $n > N_2$ 时，$|z_n - a| < \epsilon \Rightarrow z_n < a + \epsilon$。
取 $N = \max\{n_0, N_1, N_2\}$，当 $n > N$ 时：
$$a - \epsilon < y_n \le x_n \le z_n < a + \epsilon \Rightarrow |x_n - a| < \epsilon$$
得证。

### 2. 函数形式
若在 $x_0$ 的某去心邻域内满足 $g(x) \le f(x) \le h(x)$，且 $\lim_{x \to x_0} g(x) = A, \lim_{x \to x_0} h(x) = A$，则 $\lim_{x \to x_0} f(x) = A$。

## 二、第一个重要极限

### 公式：$\lim_{x \to 0} \frac{\sin x}{x} = 1$

**证明（几何法）**：
利用单位圆面积/长度关系：
在 $x \in (0, \frac{\pi}{2})$ 时，有 $\sin x < x < \tan x$。
不等式同除以 $\sin x$ 得：$1 < \frac{x}{\sin x} < \frac{1}{\cos x} \Rightarrow \cos x < \frac{\sin x}{x} < 1$。
由于 $\lim_{x \to 0} \cos x = 1, \lim_{x \to 0} 1 = 1$，由夹逼准则得证。

**变形应用**：
1. $\lim_{x \to 0} \frac{\tan x}{x} = 1$
2. $\lim_{x \to 0} \frac{\arcsin x}{x} = 1$
3. $\lim_{x \to 0} \frac{1 - \cos x}{x^2} = \frac{1}{2}$

## 三、准则 II：单调有界准则

**表述**：单调有界数列必有极限。
- **逻辑关系**：
  - 收敛 $\Rightarrow$ 有界（必要不充分）
  - 单调 + 有界 $\Leftrightarrow$ 收敛（充分必要）
  - 有界但非单调 $\nRightarrow$ 收敛（如 $\{(-1)^n\}$）

## 四、第二个重要极限

### 公式：$\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$

**证明要点**：
1. 设 $x_n = (1 + \frac{1}{n})^n$，利用二项式定理展开：
   $x_n = 1 + n \cdot \frac{1}{n} + \frac{n(n-1)}{2!} \frac{1}{n^2} + \dots + \frac{n!}{n!} \frac{1}{n^n}$
   $= 1 + 1 + \frac{1}{2!}(1 - \frac{1}{n}) + \frac{1}{3!}(1 - \frac{1}{n})(1 - \frac{2}{n}) + \dots$
2. 证明**单调性**：比较 $x_n$ 与 $x_{n+1}$，各项均增大且项数变多，故 $x_n$ 单调递增。
3. 证明**有界性**：$x_n < 1 + 1 + \frac{1}{2!} + \dots + \frac{1}{n!} < 1 + 1 + \frac{1}{2} + \frac{1}{2^2} + \dots = 3$。
4. 根据单调有界准则，极限存在，记为 $e \approx 2.718$。

**推广形式 ($1^\infty$ 型极限)**：
1. $\lim_{x \to 0} (1+x)^{1/x} = e$
2. $\lim_{f(x) \to 1, g(x) \to \infty} [f(x)]^{g(x)} = e^{\lim [f(x)-1] \cdot g(x)}$

## 五、实战总结与技巧

### 1. 夹逼准则处理分式求和
**例**：求 $\lim_{n \to \infty} \left( \frac{n}{n^2+1} + \frac{n}{n^2+2} + \dots + \frac{n}{n^2+n} \right)$
- 放大：$n \cdot \frac{n}{n^2+1} \to 1$
- 缩小：$n \cdot \frac{n}{n^2+n} \to 1$
- 结论：原极限 = 1。

### 2. “抓大头”准则（针对 $a^n + b^n + c^n$）
**结论**：$\lim_{n \to \infty} (a^n + b^n + c^n)^{1/n} = \max\{a, b, c\}$。
- **证明思路**：设 $c$ 最大，则 $c < (a^n+b^n+c^n)^{1/n} < (3c^n)^{1/n} = 3^{1/n} \cdot c \to c$。

### 3. $1^\infty$ 型快速公式
对于 $\lim (\frac{ax+b}{ax+c})^{dx}$ 型：
- 结果为 $e^{\frac{d(b-c)}{a}}$。

### 4. 递归数列求极限（单调有界准则应用）
**步骤**：
1. 证明有界（常用数学归纳法）。
2. 证明单调（作差 $a_{n+1}-a_n$ 或利用导数单调性）。
3. 设极限为 $L$，对递推式取极限解方程。
**例**：$a_1=2, a_{n+1}=\sqrt{2+a_n}$。解得 $L^2=2+L \Rightarrow L=2$。

## 六、柯西极限存在准则（Cauchy Criterion）

**定义**：数列 $\{x_n\}$ 收敛的充要条件是：
$\forall \epsilon > 0, \exists N > 0$，使得当 $m, n > N$ 时，恒有 $|x_n - x_m| < \epsilon$。
