## 7-7 常系数齐次线性微分方程

### 一、 方程形式
$$y'' + py' + qy = 0$$
其中 $p, q$ 为常数。

### 二、 求解思路
1.  **基本思想：** 寻找两个线性无关的特解 $y_1, y_2$。
2.  **尝试解：** 设 $y = e^{rx}$，代入原方程得：
    $$(r^2 + pr + q) e^{rx} = 0$$
    由于 $e^{rx} \neq 0$，故：
3.  **特征方程 (Characteristic Equation)：**
    $$r^2 + pr + q = 0$$

### 三、 解的情况分类讨论（二阶）

#### 4.① 两个不相等的实根 ($\Delta = p^2 - 4q > 0$)
*   若特征方程有两个不同实根 $r_1 \neq r_2$，则：
    $y_1 = e^{r_1x}, y_2 = e^{r_2x}$（线性无关）。
*   **通解为：**
    $$y = C_1 e^{r_1x} + C_2 e^{r_2x}$$

**例1. 求通解：$y'' - 2y' - 3y = 0$**
*   解：特征方程为 $r^2 - 2r - 3 = 0 \Rightarrow (r-3)(r+1) = 0$。
*   得到 $r_1 = 3, r_2 = -1$。
*   **通解：** $y = C_1 e^{3x} + C_2 e^{-x}$。

---

#### 4.② 两个相等的实根 ($\Delta = p^2 - 4q = 0$)
*   特征根为 $r_1 = r_2 = -\frac{p}{2}$。此时只有一个特解 $y_1 = e^{r_1x}$。
*   **推导第二个解 $y_2$：**
    利用**参数变易法**，设 $y_2 = u(x)y_1 = u(x) e^{r_1x}$。
    求导：
    $y_2' = (u' + r_1u)e^{r_1x}$
    $y_2'' = (u'' + 2r_1u' + r_1^2u)e^{r_1x}$
    代入方程 $y'' + py' + qy = 0$：
    $$e^{r_1x} [u'' + (2r_1 + p)u' + (r_1^2 + pr_1 + q)u] = 0$$
    由于 $r_1$ 是重根，满足 $2r_1 + p = 0$ 且 $r_1^2 + pr_1 + q = 0$，上式简化为：
    $$u'' = 0$$
    解得 $u(x) = Ax + B$，取最简形式 $u(x) = x$，则 $y_2 = x e^{r_1x}$。
*   **通解为：**
    $$y = (C_1 + C_2x) e^{r_1x}$$

**例2. 求特解：$y'' + 2y' + y = 0, y(0)=4, y'(0)=-2$**
*   解：特征方程 $r^2 + 2r + 1 = 0 \Rightarrow (r+1)^2 = 0 \Rightarrow r = -1$ (重根)。
*   通解：$y = (C_1 + C_2x) e^{-x}$。
*   求导：$y' = -C_1 e^{-x} + C_2(1-x)e^{-x}$。
*   代入初值：
    $\begin{cases} C_1 = 4 \\ -C_1 + C_2 = -2 \end{cases} \Rightarrow \begin{cases} C_1 = 4 \\ C_2 = 2 \end{cases}$
*   **特解：** $y = (4 + 2x)e^{-x} = 4e^{-x} + 2xe^{-x}$。

---

#### 4.③ 一对共轭复根 ($\Delta = p^2 - 4q < 0$)
*   特征根为 $r_{1,2} = \alpha \pm \beta i$（其中 $\alpha = -\frac{p}{2}, \beta = \frac{\sqrt{4q-p^2}}{2}$）。
*   形式解：$y_1 = e^{(\alpha + \beta i)x}, y_2 = e^{(\alpha - \beta i)x}$。
*   利用**欧拉公式** $e^{i\theta} = \cos\theta + i\sin\theta$：
    $y_1 = e^{\alpha x}(\cos\beta x + i\sin\beta x)$
    $y_2 = e^{\alpha x}(\cos\beta x - i\sin\beta x)$
*   取实部和虚部构造实值解：
    $\bar{y}_1 = \frac{1}{2}(y_1 + y_2) = e^{\alpha x} \cos\beta x$
    $\bar{y}_2 = \frac{1}{2i}(y_1 - y_2) = e^{\alpha x} \sin\beta x$
*   **通解为：**
    $$y = e^{\alpha x} (C_1 \cos\beta x + C_2 \sin\beta x)$$

**例3. 求通解：$y'' - 2y' + 5y = 0$**
*   解：特征方程 $r^2 - 2r + 5 = 0 \Rightarrow r = \frac{2 \pm \sqrt{-16}}{2} = 1 \pm 2i$。
*   这里 $\alpha = 1, \beta = 2$。
*   **通解：** $y = e^x (C_1 \cos 2x + C_2 \sin 2x)$。

---

### 四、 高阶常系数齐次线性微分方程

#### 1. 三阶方程：$y''' + py'' + qy' + ty = 0$
*   特征方程：$r^3 + pr^2 + qr + t = 0$。
*   **根的情况讨论：**
    1.  三个不同实根 $r_1, r_2, r_3 \Rightarrow y = C_1 e^{r_1x} + C_2 e^{r_2x} + C_3 e^{r_3x}$。
    2.  有二重根 $r_1 = r_2, r_3 \Rightarrow y = (C_1 + C_2x) e^{r_1x} + C_3 e^{r_3x}$。
    3.  有三重根 $r_1 = r_2 = r_3 \Rightarrow y = (C_1 + C_2x + C_3x^2) e^{r_1x}$。
    4.  有一对复根 $r_{1,2} = \alpha \pm \beta i$ 及一实根 $r_3 \Rightarrow y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x) + C_3 e^{r_3x}$。

**补例1. $y''' - y' = 0$ 求通解**
*   解：$r^3 - r = 0 \Rightarrow r(r+1)(r-1) = 0$。
*   根为 $0, -1, 1$。
*   **通解：** $y = C_1 + C_2 e^{-x} + C_3 e^x$。

**补例2. $y''' - 3y'' + 3y' - y = 0$ 求通解**
*   解：特征方程为 $(r-1)^3 = 0 \Rightarrow r = 1$ (三重根)。
*   **通解：** $y = (C_1 + C_2x + C_3x^2) e^x$。

---

#### 2. n 阶方程：$y^{(n)} + p_1 y^{(n-1)} + \dots + p_{n-1} y' + p_n y = 0$
*   特征方程：$r^n + p_1 r^{n-1} + \dots + p_n = 0$。
*   **通解构成规则：**
    *   **单实根 $r$：** 对应项 $Ce^{rx}$。
    *   **k 重实根 $r$：** 对应项 $(C_1 + C_2x + \dots + C_k x^{k-1})e^{rx}$。
    *   **单对共轭复根 $\alpha \pm \beta i$：** 对应项 $e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$。
    *   **k 重共轭复根：** 对应项 $e^{\alpha x} [(C_1 + \dots + C_k x^{k-1})\cos\beta x + (D_1 + \dots + D_k x^{k-1})\sin\beta x]$。

**例6. 求通解：$y^{(4)} - 2y''' + 5y'' = 0$**
*   解：$r^4 - 2r^3 + 5r^2 = 0 \Rightarrow r^2(r^2 - 2r + 5) = 0$。
*   特征根：$r_{1,2} = 0$ (二重实根)，$r_{3,4} = 1 \pm 2i$ (单对复根)。
*   **通解：** $y = (C_1 + C_2x) + e^x (C_3 \cos 2x + C_4 \sin 2x)$。

**补例. 某常系数齐次线性方程特征方程如下，求通解：**
$$[(r-3)^2 + 4]^3 (r-2)^3 (r+1) = 0$$
*   解：
    1.  根 $r = -1$ (单根) $\Rightarrow C_1 e^{-x}$。
    2.  根 $r = 2$ (三重根) $\Rightarrow (C_2 + C_3x + C_4x^2) e^{2x}$。
    3.  根 $r = 3 \pm 2i$ (三重复根) $\Rightarrow e^{3x} [(C_5 + C_6x + C_7x^2)\cos 2x + (C_8 + C_9x + C_{10}x^2)\sin 2x]$。
*   **最终通解：**
    $$y = C_1 e^{-x} + (C_2 + C_3x + C_4x^2)e^{2x} + e^{3x} [(C_5 + C_6x + C_7x^2)\cos 2x + (C_8 + C_9x + C_{10}x^2)\sin 2x]$$

## 个人总结与思考 (Summary & Reflection)

### 1. 核心映射关系
这部分内容的本质是**将微分运算转化为代数运算**。通过设 $y = e^{rx}$，导数阶数直接对应了代数方程的次数。
*   **微分算子 $D$：** 可以理解为 $r$。
*   **线性性质：** 保证了我们可以通过线性组合特征根对应的基解来构造通解。

### 2. 关于重根的直观理解
当出现重根时，简单的 $e^{rx}$ 无法提供足够的“自由度”（线性无关解的数量必须等于方程阶数）。引入 $x, x^2 \dots$ 的乘子，本质上是在解空间中寻找新的正交方向，这种处理方式在拉普拉斯变换和矩阵对角化（Jordan标准型）中都有深刻的对应。

### 3. 复根的物理意义
复根总是成对出现，且通解中出现了三角函数。这对应了物理学中的**简谐振动或阻尼振动**。$\alpha$ 决定了振幅的衰减或增长，而 $\beta$ 决定了振动的频率。
