# 笔记：7-3 齐次微分方程



## 一、形式
$$y' = \frac{dy}{dx} = \varphi\left(\frac{y}{x}\right)$$

**例如**：$(xy - x^2)dx - (x^2 - 2xy)dy = 0$
$\Rightarrow \frac{dy}{dx} = \frac{xy - x^2}{x^2 - 2xy} = \frac{\frac{y}{x} - 1}{1 - 2\frac{y}{x}} = \varphi\left(\frac{y}{x}\right)$

---

## 二、解法
1. **变量替换**：令 $u = \frac{y}{x} \Rightarrow y = ux \Rightarrow y' = u + x \frac{du}{dx}$
2. **代入方程**：$u + x \frac{du}{dx} = \varphi(u)$
3. **分离变量**：$x \frac{du}{dx} = \varphi(u) - u \Rightarrow \frac{dx}{x} = \frac{du}{\varphi(u) - u}$
4. **积分还原**：积分求出 $u$ 与 $x$ 的关系后，将 $u$ 还原为 $\frac{y}{x}$。

---

## 三、例题提取

### 例 1：求通解 $y^2 + x^2 \frac{dy}{dx} = xy \frac{dy}{dx}$

- **变形**：$\frac{dy}{dx} = \frac{y^2}{xy - x^2} = \frac{(\frac{y}{x})^2}{(\frac{y}{x}) - 1}$
- **替换**：令 $u = \frac{y}{x}$，则 $\frac{dy}{dx} = u + x \frac{du}{dx} = \frac{u^2}{u-1}$
- **整理**：$x \frac{du}{dx} = \frac{u^2}{u-1} - u = \frac{u^2 - (u^2 - u)}{u-1} = \frac{u}{u-1}$
- **积分**：$\int \frac{dx}{x} = \int \frac{u-1}{u} du \Rightarrow \ln x = u - \ln u + C$
- **还原**：$\ln x + \ln \frac{y}{x} = \frac{y}{x} + C \Rightarrow \ln y = \frac{y}{x} + C$
- **结果**：$y = C e^{\frac{y}{x}}$
  > **注（蓝色手写体）**：如果写成 $Cy = e^{\frac{y}{x}}$，需注意 $C \neq 0$。

---

### 补例 1：求特解 $\frac{dy}{dx} = \frac{y}{x} - \frac{1}{2}\left(\frac{y}{x}\right)^3, y(1)=1$

- **替换**：令 $u = \frac{y}{x}$，则 $\frac{dy}{dx} = u + x \frac{du}{dx} = u - \frac{1}{2}u^3$
- **整理**：$x \frac{du}{dx} = -\frac{1}{2}u^3 \Rightarrow -2 \frac{du}{u^3} = \frac{dx}{x}$
- **积分**：$\frac{1}{u^2} = \ln x + C \Rightarrow \frac{x^2}{y^2} = \ln x + C$
- **特解条件**：代入 $x=1, y=1$ $\Rightarrow 1 = 0 + C \Rightarrow C=1$
- **结果**：$y^2 = \frac{x^2}{\ln x + 1}$

---

### 补例 2：$y^3 dx + 2(x^2 - xy^2) dy = 0$，求通解

- **变形**：$\frac{dy}{dx} = \frac{y^3}{2(xy^2 - x^2)}$
- **变量替换（技巧）**：令 $x = u^2 \Rightarrow dx = 2u du$
- **代入**：$y^3 \cdot 2u du + 2(u^4 - u^2 y^2) dy = 0$
  $\Rightarrow \frac{dy}{du} = \frac{2u y^3}{2(u^2 y^2 - u^4)} = \frac{u y^3}{u^2 y^2 - u^4}$
- **齐次化**：$\frac{dy}{du} = \frac{\frac{y^3}{u^3}}{\frac{y^2}{u^2} - 1} = \frac{(\frac{y}{u})^3}{(\frac{y}{u})^2 - 1}$
- **二次替换**：令 $t = \frac{y}{u}$，则 $\frac{dy}{du} = t + u \frac{dt}{du} = \frac{t^3}{t^2-1}$
- **整理**：$u \frac{dt}{du} = \frac{t^3}{t^2-1} - t = \frac{t^3 - (t^3 - t)}{t^2-1} = \frac{t}{t^2-1}$
- **积分**：$\int \frac{t^2-1}{t} dt = \int \frac{du}{u} \Rightarrow \frac{1}{2}t^2 - \ln t = \ln u + \ln C$
- **还原**：$\frac{y^2}{2u^2} - \ln \frac{y}{u} = \ln u + \ln C \Rightarrow \frac{y^2}{2u^2} - \ln y + \ln u = \ln u + \ln C$
- **由于 $u^2 = x$**：$\frac{y^2}{2x} = \ln y + \ln C$
- **最终结果**：$\frac{y^2}{2x} = \ln(Cy)$
