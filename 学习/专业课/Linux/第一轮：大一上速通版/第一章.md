### **Linux Chapter 1: Identifying Linux as an Operating System**

#### **1. 什么是操作系统？(What is an OS?)**

想象你在开一辆车：
*   **硬件 (Hardware)**：是发动机、轮子、底盘。
*   **用户 (User)**：是你，你想去某个地方。
*   **操作系统 (OS)**：就是**方向盘、油门、仪表盘和车载电脑**。你不能直接用脚去踹发动机（直接操作硬件），你需要通过方向盘（OS）来控制车。

**定义 (Exam Definition):**
> An **Operating System (OS)** is software that acts as an **interface** between a **user** and **computer hardware**. It manages hardware resources, system memory, and user interaction.

---

### **2. 操作系统的类型 (Types of OS)**
*考点：对比 MS-DOS 和 Linux 的区别。*

| 典型例子 (Examples)                     | 类型 (Type)               | 描述 (Description)                     |
| :---------------------------------- | :---------------------- | :----------------------------------- |
| **MS-DOS**, Windows 95/98           | **单用户系统 (Single-user)** | 一次只能一个人用。                            |
| **Linux**, **UNIX**, Windows Server | **多用户系统 (Multiuser)**   | 允许多个用户**同时 (Simultaneously)** 登录和工作。 |

**必考对比题：**
*   **MS-DOS**: Single-user, Single-tasking (一次干一件事).
*   **Linux**: **Multiuser (多用户)**, **Multitasking (多任务)**.

### **3. Linux 的历史 (History Timeline)**
*这里不需要背年份，但必须记住几个**人名**和**关系**。外教特别喜欢考这个文化背景。*

1.  **UNIX (祖师爷)**:
    *   **1969年**，诞生于 **Bell Labs (贝尔实验室)**。
    *   作者：**Ken Thompson** & **Dennis Ritchie** (记住这两个名字，尤其是 Dennis Ritchie，也是 C 语言之父)。
    *   **1973年**：用 **C 语言** 重写了 UNIX。这很重要，因为 C 语言的可移植性让 UNIX 能跑在各种机器上。

2.  **GNU Project (精神领袖)**:
    *   **1983年**，由 **Richard Stallman** 发起。
    *   目标：创建一个完全**自由 (Free/Open Source)** 的操作系统。
    *   协议：**GPL (General Public License)**。它规定了源码必须公开。

3.  **Linux (内核诞生)**:
    *   **1991年**，芬兰大学生 **Linus Torvalds** (记住他！) 写出了 **Linux Kernel (内核)**。
    *   **关键公式**：**Linux = Linux Kernel + GNU Tools**。
    *   吉祥物：**Tux** (那只企鹅)。
    * POSIX: Portable Operating System Interface - standards for Unix-like OS compatibility

### **4. Linux 的架构 (Architecture)**
*这是本章最核心的知识点，通常会出一张图让你填空。*

Linux 系统像一个洋葱，从内到外分为三层：

1.  **Kernel (内核)**: **The Core.**
    *   它直接管理硬件（CPU、内存、硬盘）。
    *   它是系统的“心脏”。

2.  **Shell (外壳)**: **The CLI (Command Line Interface).**
    *   它是用户和内核之间的“翻译官”。
    *   你输入的命令（如 `ls`, `cd`），Shell 翻译给内核听。
    *   常见的 Shell：**bash** (默认), sh, zsh。

3.  **Utilities & Applications (工具与应用)**:
    *   最外层，比如文本编辑器 (vi)、网页服务器 (Apache)、编译器 (gcc)。

---

### **5. 文件系统结构 (File System Structure)**
*Windows 有 C 盘、D 盘，Linux 没有盘符的概念。*

*   **Tree-like Structure (树状结构)**: 一切都从 **Root (根目录 `/`)** 开始。
*   **Everything is a file**: 在 Linux 眼里，键盘、鼠标、硬盘，统统都是文件。

**必背的关键目录 (Key Directories):**
*   **`/`**: Root directory (根目录).
*   **`/bin`**: **Bin**ary utilities (存放常用命令，如 ls, cp).
*   **`/home`**: Users' home directories (普通用户的家，如 /home/student).
*   **`/etc`**: System configuration files (配置文件，如密码文件).
*   **`/var`**: **Var**iable data (经常变动的数据，如日志 logs).

#### **补充：**

#### **1. Multiuser System Components (多用户系统组件)**
*PDF Page 1 中特有的考点，容易被忽略。*

*   **Server (服务器)**: 中央计算机，负责执行客户端的请求。
*   **Terminal (终端)**: 用户和服务器之间的接口（显示器+键盘）。
    *   **Dumb Terminal (哑终端)**: 只有显示器和键盘，**没有 CPU 和硬盘**。它完全依赖服务器计算。
    *   **Smart Terminal (智能终端)**: 有自己的 CPU 和外设（像现在的 PC）。
    *  **Console (控制台)**：直接连接到主机（Server）上的那套显示器和键盘。
        - 理解： 它是系统管理员用的“主终端”
    - **Pseudo-Terminal (伪终端)**：这是我们在图形界面（GUI）下打开的那个“黑框框”（Terminal Window）。它不是物理设备，是软件模拟的。
        
        - 考点： 现在的 Linux 都是在用伪终端（pts）。

### **2. 必须掌握的 Chapter 1 基础命令**
*PDF Page 3 中列出的 "Basic Linux Commands"，这是第一章的实操重点。*

这些命令通常用于**获取系统状态**或**控制屏幕**。

#### **A. System Information (系统信息)**
*   **`who`**:
    *   **作用**: Show logged-in users (显示谁正在登录系统)。
    *   *记忆技巧*: "Who is there?"
*   **`uptime`**:
    *   **作用**: Show system runtime and load (显示系统运行了多久，以及负载情况)。
*   **`date`**:
    *   **作用**: Display date and time (显示日期时间)。
    *   **高阶考点 (Formatting)**: PDF 特别提到了格式化（Page 3 底部黄色框）：
        *   `date "+%T"` -> 只显示时间 (Time)
        *   `date "+%D"` -> 只显示日期 (Date)
        * %T = Time (HH:MM:SS)
		- %D = Date (mm/dd/yy) —— **记住这个反直觉的点！**
*   **`passwd`**:
    *   **作用**: Change password (修改密码)。*注意拼写是 passwd 不是 password。*

#### **B. Screen Control (屏幕控制)**
*   **`clear`**:
    *   **作用**: Clear terminal screen (清屏)。
*   **`tput` 命令集** (这个比较偏，但 PDF 里有表格，必须看)：
    *   `tput clear`: 同 clear。
    *   `tput cup row col`: Move cursor to position (把光标移动到第几行第几列)。*例如 `tput cup 10 20`*。
    *   `tput smso` / `rmso`: Start/End **Reverse Video** Mode (开启/关闭反白显示模式)。
    *   `tput blink`: 开启闪烁文字。

#### **C. Help Commands (求助命令)**
*   **`man command`**: Display **manual** pages (显示详细的手册)。
*   **`info command`**: Detailed documentation (比 man 更详细的文档)。

### **3. 会话管理 (Starting & Ending Sessions)**
*PDF Page 4*

#### **A. Login (登录)**
*   **Root User**: 超级管理员，提示符通常是 **`#`**。
*   **Normal User**: 普通用户，提示符通常是 **`$`**。
*   用户信息存在哪？
    *   `/etc/passwd` (存用户信息)
    *   `/etc/shadow` (存加密后的密码)

#### **B. Ending Session (退出/关机)**
*   **退出登录**:
    *   `logout` 或 `exit` 命令。
    *   快捷键：**`Ctrl + D`** (这是 EOF，常用快捷键)。
*   **关机 (Shutdown)**:
    *   `sudo shutdown -h now` (立即关机)。
*   **重启 (Reboot)**:
    *   `sudo reboot`。
