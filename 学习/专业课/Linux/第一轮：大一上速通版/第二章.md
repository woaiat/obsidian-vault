### **Linux Chapter 2: Identifying the Basics of Linux**

#### **1. Linux File System (文件系统结构)**
*PDF Page 5*

*   **Tree-like (树状结构)**: 一切起点都是 **Root (根目录 `/`)**。
* **home目录**：~  **本级目录**：.  **上级目录**：..
*   **Pathnames (路径名)**:
    *   **Absolute (绝对路径)**: Must start with `/` (e.g., `/home/student`).
    *   **Relative (相对路径)**: Relative to current directory (e.g., `Documents/file.txt`).
    *   **Tilde (`~`)**: Represents the user's **home directory**.

1. **. 和 .. 的真身**：
    
    - 每个目录下都有两个隐藏文件：
        
        - **. (Dot)**: 代表 **Current Directory (当前目录)**。
            
            - 用法： ./script.sh (运行当前目录下的脚本)。
                
        - **.. (Dot Dot)**: 代表 **Parent Directory (父目录/上一级)**。
            
            - 用法： cd .. (返回上一级)。
                
2. **波浪号 ~ 的展开**：
    
    - cd ~ = 回到当前用户的家。
        
    - **cd ~steve** (注意这个用法) = 去用户 **steve** 的家目录 (/home/steve)。
        
    - 考点： 填空题可能会问 cd ~root 去哪里？答案是 /root。
        

---
**Key Directories (核心目录表 - 必背)**:
*   `/bin`: **Essential binary** utilities (常用命令如 ls, cp).
*   `/dev`: **Device** files (硬件设备文件).
*   `/etc`: System **configuration** files (配置文件).
*   `/home`: User home directories (普通用户家目录).
*   `/var`: **Variable** data (logs, spool) (经常变动的数据).
*   `/lib`: System libraries (库文件).

---

#### **2. Types of Files (文件类型)**
*PDF Page 5 - 6 (考点：Linux 中一切皆文件)*

1.  **Ordinary Files (普通文件)**: Data, text, programs. (符号 `-`)
2.  **Directory Files (目录文件)**: Contain lists of files. (符号 `d`)
3.  **Special Files (特殊文件)**:
    *   **Character Device**: Read/write one character at a time (e.g., Modem, Keyboard).
    *   **Block Device**: Read/write blocks of data (e.g., Hard Disk).
      
4.  **Links (链接)**:
    *   **Hard Links**: Multiple names for the same file.
    *   **Symbolic Links (Soft Links)**: Pointers to files (like Shortcuts in Windows).

---

#### **3. Types of Users (用户类型)**
*PDF Page 6*

1.  **System Administrator (root)**: Full access.
2.  **File Owner**: The user who created the file.
3.  **Group Owner**: Users belonging to the same group.
4.  **Other Users**: Everyone else.

---

#### **4. Directory Commands (目录操作命令)**
*PDF Page 6 (实操核心)*

*   **`pwd`**: Print Current Directory (显示当前路径).
*   **`cd`**: Change Directory.
    *   `cd /home/user` (绝对路径跳转)
    *   `cd -` (返回到上一次的路径)
*   **`mkdir`**: Make Directory (创建目录).
    *   `mkdir newdir` ：
    * -p:如果该目录不存在 就会先创造该目录
*   **`rmdir`**: Remove Directory.
    *   **限制**: Only removes **EMPTY** directories (只能删除空目录).
*   **`ls`**: List directory contents.

**Common `ls` Options (ls 的参数 - 必考匹配题):**
*   **`-l`**: **Long** listing (显示详细信息：权限、所有者、大小、时间).
*   **`-a`**: **All** files (显示隐藏文件，即以 `.` 开头的文件).
*   **`-F`**: File type indicators (显示文件类型符号，如目录后加 `/`，可执行程序加*，链接文件加@).
*   **`-R`**: **Recursive** listing (递归显示子目录).
*   **`-r`**: **Reverse** order (倒序排列).
*   **`-S`**: Sort by **File Size** (按大小排序).

---

#### **5. File Commands (文件操作命令)**
*PDF Page 7*

*   **`cat`**: Display file content (查看文件内容).
*   **`cp`**: Copy files.
    *   `cp source dest`
    *   Option **`-r`**: Copy **recursively** (必用于复制目录!).
*   **`mv`**: Move or **Rename** (移动或==重命名==).
    *   `mv oldname newname`
    *   Option **`-i`**: Interactive (覆盖前提示).
*   **`rm`**: Remove files.
    *   Option **`-r`**: Recursive (删除目录必须加).
    *   Option **`-f`**: **Force** (强制删除，不提示).
*   **`more`**: Pagewise display (只能向前翻页).
*   **`less`**: Pagewise display (可以向前向后翻页，比 more 强).

---

#### **6. Wildcard Characters (通配符)**
*PDF Page 7 (重点：文件名匹配)*

*   **`*` (Asterisk)**: Matches **any string** (including empty).
    *   `ls *.txt` (列出所有 txt 文件).
*   **`?` (Question Mark)**: Matches **exactly one** character.
    *   `ls file?.txt` (匹配 file1.txt, fileA.txt, 但不匹配 file10.txt).
*   **`[ ]` (Brackets)**: Matches **one character from a set**.
    *   `ls file[123].txt` (匹配 file1.txt, file2.txt, file3.txt).

#### **7. File Access Permissions (权限管理 - 难点)**
*PDF Page 7 - 8 (大题必考区域)*

**Permissions (三种权限):**
*   **`r` (Read)**:
    *   文件: View/Copy content.
    *   目录: **List contents** (可以用 ls).
*   **`w` (Write)**:
    *   文件: Modify content.
    *   目录: **Create/Delete files** inside it.
*   **`x` (Execute)**:
    *   文件: Run as program.
    *   目录: **Access/Enter** (可以用 cd 进入).
    * ==如果没有x只有w 也是没有办法读写文件的！！ 相当于w权限被废除了！！==

**Changing Permissions: `chmod` (Change Mode)**
*PDF Page 8 提供了两种模式，都要会。*

**A. Symbolic Mode (符号模式)**
*   Who: `u` (user), `g` (group), `o` (others), `a` (all).
*   Action: `+` (add), `-` (remove), `=` (set).
*   *Example*: `chmod u+x file` (给 owner 加执行权限).
*   *Example*: `chmod g-w file` (给 group 减写入权限).

**B. Absolute/Numeric Mode (数字模式 - 计算题)**
*   **Values**:
    *   **Read (r) = 4**
    *   **Write (w) = 2**
    *   **Execute (x) = 1**
*   *Example*: **`chmod 755 file`**
    *   Owner: 7 = 4+2+1 (rwx)
    *   Group: 5 = 4+0+1 (r-x)
    *   Others: 5 = 4+0+1 (r-x)
*   *Example*: **`chmod 644 file`** (rw-r--r--).

**ls -l 显示的第一列第一个字符决定文件类型：**

- **-** : 普通文件
    
- **d** : 目录 (**d**irectory)
    
- **l** : 链接 (**l**ink)
