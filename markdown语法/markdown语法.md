# 📝 Markdown 语法入门笔记

## 1. 标题 (Headings)
使用 `#` 号来表示标题，1个 `#` 是一级标题，2个是二级标题，以此类推（建议在 `#` 后加一个空格）。
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```

## 2. 字体格式 (Text Styling)
*   **加粗**：左右各两个 `*`
*   *斜体*：左右各一个 `*`
*   ***斜体加粗***：左右各三个 `*`
*   ~~删除线~~：左右各两个 `~`

```markdown
**这是加粗**
*这是斜体*
***这是斜体加粗***
~~这是删除线~~
```

## 3. 列表 (Lists)
### 无序列表
使用 `-`、`+` 或 `*` 加上空格。
- 项目一
- 项目二

### 有序列表
数字加上 `.` 加上空格。
1. 第一步
2. 第二步

### 任务列表（打卡框）
- [ ] 未完成任务
- [x] 已完成任务

## 4. 引用 (Blockquotes)
使用 `>` 符号。
> 这是一个引用。
> “路漫漫其修远兮，吾将上下而求索。”

## 5. 代码 (Code)
### 行内代码
使用反引号（键盘 Tab 键上方那个键）包裹。
比如：我觉得 `Markdown` 真的很好用。

### 代码块
使用三个反引号包裹，并可以标注语言名称（如 python, sql, js 等）实现高亮。
\```python
print("Hello World")
\```

## 6. 超链接与图片 (Links & Images)
### 超链接
语法：`[显示文本](链接地址)`
[百度一下](https://www.baidu.com)

### 图片
语法：`![图片描述](图片链接地址)`
![图片提示语](https://example.com/image.jpg)

## 7. 表格 (Tables)
使用 `|` 分隔单元格，使用 `-` 分隔表头。
```markdown
| 姓名 | 年龄 | 职业 |
| :--- | :--: | ---: |
| 张三 | 25 | 程序员 |
| 李四 | 30 | 设计师 |
```
*(注：`:---` 左对齐，`:--:` 居中，`---:` 右对齐)*

## 8. 分割线 (Horizontal Rules)
使用三个以上的 `*` 或 `-`。
---

## 💡 小贴士：
1.  **空格很重要**：大多数语法符号（如 `#`, `-`, `1.`）后面都要加一个空格。
2.  **空行很重要**：如果你发现段落没有换行，尝试在段落之间多加一个空行。
3.  **常用工具**：
    *   **Typora**：目前公认最好用的 Markdown 编辑器（所见即所得）。
    *   **Obsidian / Logseq**：强大的知识库工具。
    *   **VS Code**：程序员最爱，配合插件体验极佳。
